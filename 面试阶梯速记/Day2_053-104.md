## 053. 700 二叉搜索树中的搜索【简单】
- 套路：BST｜分类：二叉搜索树 / 二叉树 / 树
- 一句话速记：BST 搜索就按大小往左或往右走。
- 面试可过：迭代从根一路下沉，`val < node.val` 去左边，否则去右边。
- 更优/补充：比 DFS 全树搜索少利用了 BST 性质就亏了。
```python
cur = root
while cur and cur.val != val:
    cur = cur.left if val < cur.val else cur.right
return cur
```

## 054. 724 寻找数组的中心下标【简单】
- 套路：前缀和｜分类：前缀和 / 数组
- 一句话速记：左边和 = 总和 - 左边和 - 当前值。
- 面试可过：先拿总和，再顺扫维护左边和，O(n)。
- 更优/补充：省掉显式前缀数组，空间 O(1)。
```python

for i, x in enumerate(nums):
    if left == total - left - x:
        return i
    left += x
return -1
```

## 055. 746 使用最小花费爬楼梯【简单】

- 一句话速记：到达当前台阶的最小花费 = 当前 cost + min(前一阶, 前两阶)。
- 面试可过：滚动 DP 即可，答案是最后两阶的较小值。
- 更优/补充：把“站在台阶上”和“到达楼顶”区分清楚。
```python

```

## 056. 872 叶子相似的树【简单】

- 一句话速记：把两棵树的叶子序列各自收集出来再比较。

```python
def leaves(node, out):

    leaves(node.left, out)
    leaves(node.right, out)
```


## 071. LCR 128 库存管理 I【简单】

```python
    else:
        r = mid

```

```python
def mirror(a, b):
```


## 083. LCR 150 彩灯装饰记录 II【简单】
- 套路：树 BFS｜分类：搜索与回溯算法 / 二叉树 / 广度优先搜索 / 树

```python
ans, q = [], deque([root])
while q:
    level = []
    for _ in range(len(q)):
        node = q.popleft()
        level.append(node.val)
        if node.left: q.append(node.left)
        if node.right: q.append(node.right)
    ans.append(level)
return ans
```

```python
    dfs(node.left)
```

## 093. LCR 176 判断是否为平衡二叉树【简单】

- 一句话速记：平衡树要后序：先拿左右高度，再顺手判断差值。
- 面试可过：后序递归返回高度，失衡时直接返回 `-1` 剪枝，O(n)。
- 更优/补充：别每个节点都单独算高度，否则会退化到 O(n^2)。
```python
def dfs(node):
    if not node: return 0
    l = dfs(node.left)
    if l == -1: return -1
    r = dfs(node.right)
    if r == -1 or abs(l - r) > 1: return -1
    return 1 + max(l, r)
return dfs(root) != -1
```

## 104. 3 无重复字符的最长子串【中等】
