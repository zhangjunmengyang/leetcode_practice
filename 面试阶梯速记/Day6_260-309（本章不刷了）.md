## 260. LCR 149 彩灯装饰记录 I【中等】
- 套路：树 BFS｜分类：搜索与回溯算法 / 二叉树 / 广度优先搜索 / 树｜原型：层序遍历（拍平）
- 一句话速记：层序遍历拍平版：队列从上到下弹，值直接按顺序收。
- 面试可过：普通 BFS，节点值按出队顺序记录即可。
- 更优/补充：要按层分组就把一层长度先记下来。
```python
ans, q = [], deque([root])
while q:
    node = q.popleft()
    ans.append(node.val)
    if node.left: q.append(node.left)
    if node.right: q.append(node.right)
return ans
```

## 261. LCR 151 彩灯装饰记录 III【中等】
- 套路：树 BFS｜分类：搜索与回溯算法 / 二叉树 / 广度优先搜索 / 树
- 一句话速记：还是层序，只是奇偶层决定是否反转。
- 面试可过：按层 BFS，偶数层正着放，奇数层倒着放。
- 更优/补充：也可用 `deque` 左右插入避免最后 reverse。
```python
ans, q, rev = [], deque([root]), False
while q:
    level = deque()
    for _ in range(len(q)):
        node = q.popleft()
        (level.appendleft if rev else level.append)(node.val)
        if node.left: q.append(node.left)
        if node.right: q.append(node.right)
    ans.append(list(level))
    rev = not rev
return ans
```

## 262. LCR 152 验证二叉搜索树的后序遍历序列【中等】
- 套路：分治｜分类：分治算法 / 二叉搜索树 / 二叉树 / 单调栈｜原型：验证 BST 后序序列
- 一句话速记：后序最后一个是根，左边都小于根，右边都大于根，再递归验证左右段。
- 面试可过：递归分段能过，平均 O(n log n)。
- 更优/补充：面试可补充单调栈倒序写法更优。
```python
def dfs(l, r):
    if l >= r: return True
    root = postorder[r]
    m = l
    while postorder[m] < root: m += 1
    for i in range(m, r):
        if postorder[i] < root: return False
    return dfs(l, m - 1) and dfs(m, r - 1)
```

## 276. LCR 185 统计结果概率【中等】
- 套路：DP｜分类：动态规划 / 数学 / 概率与统计｜原型：n 个骰子的点数概率
- 一句话速记：骰子概率题：`dp[i][s]` 表示扔 i 个骰子得到和 s 的方案数。
- 面试可过：每多一个骰子，就从上一轮所有点数和转移 1..6；最后总方案数除以 `6^n`。
- 更优/补充：滚动数组能把空间压下来，写法也不难。
```python
dp = [1] * 6
for k in range(2, n + 1):
    ndp = [0] * (5 * k + 1)
    for i, cnt in enumerate(dp):
        for face in range(1, 7):
            ndp[i + face - 1] += cnt
    dp = ndp
total = 6 ** n
return [x / total for x in dp]
```

n = len(nums)

        nums[i], nums[j] = nums[j], nums[i]
for i, x in enumerate(nums):
    if x != i + 1:
        return i + 1
