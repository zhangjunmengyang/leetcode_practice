## 105. 5 最长回文子串【中等】
- 套路：双指针｜分类：多维动态规划 / 动态规划 / 双指针 / 字符串
- 一句话速记：回文串先想中心扩展：每个点和每个缝都当中心。
- 面试可过：枚举 `2n-1` 个中心，向两边扩，O(n^2)。
- 更优/补充：更优是 Manacher，但面试大多数场景中心扩展就够。
```python
def expand(l, r):
    while l >= 0 and r < len(s) and s[l] == s[r]:
        l -= 1; r += 1
    return l + 1, r - 1
best = (0, 0)
for i in range(len(s)):
    for a, b in (expand(i, i), expand(i, i + 1)):
        if b - a > best[1] - best[0]:
            best = (a, b)
return s[best[0]:best[1] + 1]
```

## 107. 11 盛最多水的容器【中等】
- 套路：双指针｜分类：双指针 / 数组 / 贪心
- 一句话速记：面积由短板决定，所以每次只移动更短的那一边。
- 面试可过：左右夹逼，计算面积并更新答案，O(n)。
- 更优/补充：这是双指针贪心经典题。
```python
l, r, ans = 0, len(height) - 1, 0

    ans = max(ans, min(height[l], height[r]) * (r - l))
    if height[l] < height[r]:
        l += 1
    else:
        r -= 1
return ans
```

## 123. 50 Pow(x, n)【中等】
