## 001. 1 两数之和【简单】
- 套路：哈希查找｜分类：哈希 / 哈希表 / 数组
- 一句话速记：哈希表记已经扫过的数，边走边查补数。
- 面试可过：一次遍历 `seen[x] = idx`，命中 `target - x` 立刻返回，O(n)。
- 更优/补充：排序双指针会丢原索引；这题面试主解就是哈希一次遍历。
```python
seen = {}
for i, x in enumerate(nums):
    if target - x in seen:
        return [seen[target - x], i]
    seen[x] = i
```

## 002. 9 回文数【简单】
- 套路：数学｜分类：数学
- 一句话速记：只反转后一半数字，和前一半比较，避免溢出也更省事。
- 面试可过：负数直接 false；循环把末位搬到 `rev`，最后比较 `x == rev` 或 `x == rev // 10`。
- 更优/补充：字符串法能过但不是面试优先；半反转才是更稳的数学解。
```python
if x < 0 or (x % 10 == 0 and x != 0):
    return False
rev = 0
while x > rev:
    rev = rev * 10 + x % 10
    x //= 10
return x == rev or x == rev // 10
```

## 003. 13 罗马数字转整数【简单】
- 套路：字符串｜分类：数组 / 字符串 / 哈希表 / 字符串 / 数学
- 一句话速记：小值在大值左边就减，否则就加。
- 面试可过：从左到右扫，若当前值小于右边值就减掉，否则加上，O(n)。
- 更优/补充：也可从右往左维护最大值；但左到右写法更口语化，面试更顺。
```python
mp = {'I': 1, 'V': 5, 'X': 10, 'L': 50, 'C': 100, 'D': 500, 'M': 1000}
ans = 0
for i, ch in enumerate(s):
    val = mp[ch]
    if i + 1 < len(s) and val < mp[s[i + 1]]:
        ans -= val
    else:
        ans += val
return ans
```

## 006. 21 合并两个有序链表【简单】
- 套路：链表｜分类：链表 / 递归
- 一句话速记：双指针比大小，小的先接到结果链表后面。
- 面试可过：挂 dummy，`l1/l2` 谁小接谁，尾指针一路推进，O(m+n)。
- 更优/补充：递归也能写，但面试优先迭代版，更稳更省栈。
```python
dummy = tail = ListNode(0)
while l1 and l2:
    if l1.val <= l2.val:
        tail.next, l1 = l1, l1.next
    else:
        tail.next, l2 = l2, l2.next
    tail = tail.next
tail.next = l1 or l2
return dummy.next
```

## 007. 26 删除有序数组中的重复项【简单】

- 一句话速记：慢指针指向下一个要写的位置，快指针负责扫新值。
- 面试可过：有序数组去重就保留每段第一个，`nums[slow] = nums[fast]`，O(n)。
- 更优/补充：这题要原地，别额外开 set。
```python
slow = 1
for fast in range(1, len(nums)):
    if nums[fast] != nums[fast - 1]:
        nums[slow] = nums[fast]
        slow += 1
return slow
```

## 010. 35 搜索插入位置【简单】

- 一句话速记：找位置题本质就是找第一个 `>= target` 的位置。
- 面试可过：标准左边界二分，命中就收右边界，没命中最后落在插入位。
- 更优/补充：能背出这一版，后面很多 lower_bound 题都通用。
```python
l, r = 0, len(nums)
while l < r:
    mid = (l + r) // 2
    if nums[mid] >= target:
        r = mid
    else:
        l = mid + 1
return l
```

## 013. 67 二进制求和【简单】
- 套路：字符串｜分类：位运算 / 字符串 / 数学 / 模拟
- 一句话速记：从末尾往前加，和小学竖式一样，额外带一个 carry。
- 面试可过：双指针倒序 + 进位，复杂度 O(n)。
- 更优/补充：也能转十进制再转回去，但没必要。
```python
i, j, carry = len(a) - 1, len(b) - 1, 0
ans = []
while i >= 0 or j >= 0 or carry:
    s = carry
    if i >= 0: s += int(a[i]); i -= 1
    if j >= 0: s += int(b[j]); j -= 1
    ans.append(str(s % 2))
    carry = s // 2
return ''.join(reversed(ans))
```

## 014. 69 x 的平方根 【简单】
- 套路：二分｜分类：数学 / 二分查找
- 一句话速记：找最大满足 `mid * mid <= x` 的整数。
- 面试可过：整数平方根就是右边界二分，注意乘法可能溢出时改用 `mid <= x // mid`。
- 更优/补充：牛顿迭代更快，但面试讲二分最稳。
```python
l, r = 0, x
while l <= r:
    mid = (l + r) // 2
    if mid <= x // max(mid, 1):
        l = mid + 1
    else:
        r = mid - 1
return r
```

## 016. 88 合并两个有序数组【简单】
- 套路：双指针｜分类：数组 / 字符串 / 双指针 / 排序 / 数组
- 一句话速记：从后往前填，避免覆盖 `nums1` 里还没用到的值。

```python
i, j, k = m - 1, n - 1, m + n - 1
while j >= 0:
    if i >= 0 and nums1[i] > nums2[j]:
        nums1[k], i = nums1[i], i - 1
    else:
        nums1[k], j = nums2[j], j - 1
    k -= 1
```

## 017. 94 二叉树的中序遍历【简单】
- 套路：树 DFS｜分类：二叉树 / 栈 / 树 / 深度优先搜索
- 一句话速记：中序遍历模板：一路压左，弹出访问，再去右边。
- 面试可过：递归和迭代都能过；面试里写迭代更能体现你掌握遍历模板。
- 更优/补充：BST 相关题很多都复用这套中序骨架。
```python
st, cur, ans = [], root, []
while st or cur:
    while cur:
        st.append(cur)
        cur = cur.left
    cur = st.pop()
    ans.append(cur.val)
    cur = cur.right
return ans
```

```python
def mirror(a, b):

    if not a or not b or a.val != b.val: return False
    return mirror(a.left, b.right) and mirror(a.right, b.left)
```

## 020. 104 二叉树的最大深度【简单】
- 套路：树 DFS｜分类：二叉树 / 二叉树 - 深度优先搜索 / 广度优先搜索 / 树
- 一句话速记：深度题最直接：当前节点深度 = 1 + max(左深, 右深)。
- 面试可过：递归一行就能写完，O(n)。
- 更优/补充：BFS 也能数层数，但 DFS 更短。
```python
def dfs(node):

    return 1 + max(dfs(node.left), dfs(node.right))
```


## 023. 118 杨辉三角【简单】

```python
ans = []
for i in range(numRows):

```

## 024. 121 买卖股票的最佳时机【简单】

- 更优/补充：一次交易题主解就是这个 O(n) 贪心。
```python
min_price = float('inf')
ans = 0
for p in prices:
    min_price = min(min_price, p)
    ans = max(ans, p - min_price)
```


## 030. 190 颠倒二进制位【简单】
- 套路：位运算｜分类：位运算 / 分治
- 一句话速记：一位一位取出低位，再塞到答案高位方向。

- 更优/补充：固定 32 位，复杂度就是 O(1)。
```python
ans = 0
for _ in range(32):

return ans
```

```python
ans = 0
```

## 036. 222 完全二叉树的节点个数【简单】

- 面试可过：每层比较左右高度，满子树节点数可直接算，复杂度 O(log^2 n)。
- 更优/补充：普通 DFS 也能过但没吃到“完全二叉树”信息。
```python
def left_h(node):
    h = 0
    while node:
        h += 1
        node = node.left
    return h
if not root: return 0
lh, rh = left_h(root.left), left_h(root.right)
if lh == rh:
    return (1 << lh) + countNodes(root.right)
```


## 048. 530 二叉搜索树的最小绝对差【简单】

```python
ans, prev = float('inf'), None
def dfs(node):
    nonlocal ans, prev
    if not node: return
    dfs(node.left)
    if prev is not None:
        ans = min(ans, node.val - prev)
    prev = node.val
    dfs(node.right)
dfs(root)
return ans
```

- 套路：树 DFS｜分类：二叉树 / 树 / 深度优先搜索
- 一句话速记：求深度时顺便更新直径，直径 = 左深 + 右深。
- 面试可过：后序 DFS 返回深度，遍历每个节点时更新全局答案。
- 更优/补充：这是二叉树题里最典型的“返回子树值 + 更新全局值”。
```python
ans = 0
def dfs(node):
    nonlocal ans
    if not node: return 0
    l = dfs(node.left)
    r = dfs(node.right)
    ans = max(ans, l + r)
    return 1 + max(l, r)
dfs(root)
return ans
```

## 051. 637 二叉树的层平均值【简单】

- 一句话速记：层序遍历时顺手累加这一层的和，再除以层大小。
- 面试可过：BFS 一层一层弹，复杂度 O(n)。
- 更优/补充：层题看到 average / max / right view 都是这个套路。
```python
ans, q = [], deque([root])
while q:
    level_size = len(q)
    s = 0
    for _ in range(level_size):
        node = q.popleft()
        s += node.val
        if node.left: q.append(node.left)
        if node.right: q.append(node.right)
    ans.append(s / level_size)
return ans
```

## 052. 643 子数组最大平均数 I【简单】
- 套路：滑窗｜分类：滑动窗口 / 数组
- 一句话速记：定长窗口就只做一件事：右边进一个，左边出一个。
- 面试可过：先算前 `k` 个和，再滑窗更新最大和，最后除以 `k`。
- 更优/补充：定长窗口题几乎都长这样。
```python
s = sum(nums[:k])
best = s
for i in range(k, len(nums)):
    s += nums[i] - nums[i - k]
    best = max(best, s)
return best / k
```
