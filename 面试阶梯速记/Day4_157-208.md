## 166. 148 排序链表【中等】
- 套路：链表 + 归并｜分类：分治 / 链表 / 双指针 / 归并排序

- 更优/补充：中点用快慢指针找。
```python
def sort(head):
    if not head or not head.next: return head
    slow = fast = head
    pre = None
    while fast and fast.next:
        pre, slow, fast = slow, slow.next, fast.next.next
    pre.next = None
    return merge(sort(head), sort(slow))
```

            ans = max(ans, dp[i][j])
