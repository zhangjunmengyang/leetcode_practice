## 121. 买卖股票的最佳时机 【简单】

### 思路

**核心思想：维护历史最低价，计算每天卖出的最大利润**

- 遍历价格数组，维护到当前为止的最低买入价格
- 每天计算：如果今天卖出，利润 = 今天价格 - 历史最低价
- 更新最大利润

**贪心策略：总是在最低点买入，在之后的每一天尝试卖出**

### 代码

```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        min_price = float('inf')  # 历史最低价
        max_profit = 0  # 最大利润

        for price in prices:
            min_price = min(min_price, price)  # 更新最低买入价
            max_profit = max(max_profit, price - min_price)  # 更新最大利润

        return max_profit
```

## 55. 跳跃游戏 【中等】

### 思路

**核心思想：维护能到达的最远位置**

- 遍历数组，不断更新能跳到的最远位置
- 如果当前位置超过了能到达的最远位置，说明无法继续
- 如果最远位置 >= 最后一个下标，返回 True

**贪心策略：每一步都尽可能跳得更远**

### 代码

```python
class Solution:
    def canJump(self, nums: List[int]) -> bool:
        max_reach = 0  # 能到达的最远位置
        n = len(nums)

        for i in range(n):
            if i > max_reach:  # 当前位置无法到达
                return False
            max_reach = max(max_reach, i + nums[i])  # 更新最远位置
            if max_reach >= n - 1:  # 已经能到达终点
                return True

        return True
```

## 45. 跳跃游戏 II 【中等】

### 思路

**核心思想：每次跳跃覆盖的范围内，选择能跳得最远的位置**

- 维护当前跳跃的边界 `end` 和能到达的最远位置 `max_reach`
- 遍历到边界时，必须跳一次，更新边界为最远位置
- 统计跳跃次数

**贪心策略：在每一步可选范围内，选择下一步能跳得最远的**

### 代码

```python
class Solution:
    def jump(self, nums: List[int]) -> int:
        n = len(nums)
        if n == 1:
            return 0

        jumps = 0  # 跳跃次数
        end = 0  # 当前跳跃的边界
        max_reach = 0  # 能到达的最远位置

        for i in range(n - 1):  # 不需要遍历最后一个位置
            max_reach = max(max_reach, i + nums[i])  # 更新最远位置
            if i == end:  # 到达当前跳跃边界，必须跳
                jumps += 1
                end = max_reach  # 更新边界
                if end >= n - 1:  # 已能到达终点
                    break

        return jumps
```

## 763. 划分字母区间 【中等】

### 思路

**核心思想：每个字母的最后出现位置决定了区间的最小范围**

1. 预处理：记录每个字母最后出现的位置
2. 遍历字符串，维护当前区间的最远边界
3. 当遍历到区间边界时，说明区间内所有字母都不会在后面出现，可以划分

**贪心策略：尽量让每个区间包含其所有字母的完整出现范围**

### 代码

```python
class Solution:
    def partitionLabels(self, s: str) -> List[int]:
        # 记录每个字母最后出现的位置
        last = {c: i for i, c in enumerate(s)}

        result = []
        start = 0  # 当前区间起点
        end = 0  # 当前区间终点

        for i, c in enumerate(s):
            end = max(end, last[c])  # 扩展区间到包含当前字母的最后位置
            if i == end:  # 到达区间终点，可以划分
                result.append(end - start + 1)
                start = i + 1  # 开始新区间

        return result
```
