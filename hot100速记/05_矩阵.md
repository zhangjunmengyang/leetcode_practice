## 73. 矩阵置零 【中等】

### 思路

**核心：用第一行和第一列作为标记数组**

1. 先记录第一行、第一列本身是否需要置零
2. 遍历矩阵（除第一行第一列），若 `matrix[i][j] == 0`，则标记 `matrix[i][0] = 0` 和 `matrix[0][j] = 0`
3. 根据标记，将对应行列置零（从后往前处理，避免覆盖标记）
4. 最后处理第一行和第一列

**时间 O(mn)，空间 O(1)**

### 代码

```python
class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        m, n = len(matrix), len(matrix[0])

        # 记录第一行、第一列是否需要置零
        first_row_zero = any(matrix[0][j] == 0 for j in range(n))
        first_col_zero = any(matrix[i][0] == 0 for i in range(m))

        # 用第一行第一列作为标记
        for i in range(1, m):
            for j in range(1, n):
                if matrix[i][j] == 0:
                    matrix[i][0] = 0  # 标记该行
                    matrix[0][j] = 0  # 标记该列

        # 根据标记置零（不含第一行第一列）
        for i in range(1, m):
            for j in range(1, n):
                if matrix[i][0] == 0 or matrix[0][j] == 0:
                    matrix[i][j] = 0

        # 处理第一行
        if first_row_zero:
            for j in range(n):
                matrix[0][j] = 0

        # 处理第一列
        if first_col_zero:
            for i in range(m):
                matrix[i][0] = 0
```

## 54. 螺旋矩阵 【中等】

### 思路

**核心：定义四个边界，按层顺时针收缩**

1. 定义边界：`top, bottom, left, right`
2. 按顺序遍历：右 → 下 → 左 → 上
3. 每遍历完一条边，收缩对应边界
4. 当边界交叉时停止

**时间 O(mn)，空间 O(1)**（不算结果数组）

### 代码

```python
class Solution:
    def spiralOrder(self, matrix: List[List[int]]) -> List[int]:
        if not matrix:
            return []

        res = []
        top, bottom = 0, len(matrix) - 1
        left, right = 0, len(matrix[0]) - 1

        while top <= bottom and left <= right:
            # 向右：遍历上边界
            for j in range(left, right + 1):
                res.append(matrix[top][j])
            top += 1

            # 向下：遍历右边界
            for i in range(top, bottom + 1):
                res.append(matrix[i][right])
            right -= 1

            # 向左：遍历下边界（注意判断是否还有行）
            if top <= bottom:
                for j in range(right, left - 1, -1):
                    res.append(matrix[bottom][j])
                bottom -= 1

            # 向上：遍历左边界（注意判断是否还有列）
            if left <= right:
                for i in range(bottom, top - 1, -1):
                    res.append(matrix[i][left])
                left += 1

        return res
```

## 48. 旋转图像 【中等】

### 思路

**核心：先转置，再水平翻转**

顺时针旋转 90° = 转置 + 左右翻转

- **转置**：`matrix[i][j]` 与 `matrix[j][i]` 交换
- **水平翻转**：每行反转

**技巧记忆**：
- 顺时针 90°：转置 + 左右翻转
- 逆时针 90°：转置 + 上下翻转

**时间 O(n²)，空间 O(1)**

### 代码

```python
class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        n = len(matrix)

        # 第一步：转置矩阵（沿主对角线翻转）
        for i in range(n):
            for j in range(i + 1, n):  # 只遍历上三角
                matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]

        # 第二步：水平翻转每一行
        for i in range(n):
            matrix[i].reverse()
            # 或者：matrix[i] = matrix[i][::-1]
```

## 240. 搜索二维矩阵 II 【中等】

### 思路

**核心：从右上角或左下角开始搜索**

矩阵特点：每行从左到右递增，每列从上到下递增

**从右上角 (0, n-1) 开始**：
- 当前值 > target：左移（排除当前列）
- 当前值 < target：下移（排除当前行）
- 相等则找到

**为什么选右上角？** 因为它是行最大、列最小的位置，可以决定唯一方向

**时间 O(m+n)，空间 O(1)**

### 代码

```python
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        if not matrix or not matrix[0]:
            return False

        m, n = len(matrix), len(matrix[0])

        # 从右上角开始
        i, j = 0, n - 1

        while i < m and j >= 0:
            if matrix[i][j] == target:
                return True
            elif matrix[i][j] > target:
                j -= 1  # 当前值太大，左移
            else:
                i += 1  # 当前值太小，下移

        return False
```
