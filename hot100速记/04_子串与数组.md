## 53. 最大子数组和 【中等】

### 思路

**动态规划 / Kadane算法**

核心思想：对于每个位置，要么延续之前的子数组，要么从当前位置重新开始。

状态转移：`dp[i] = max(dp[i-1] + nums[i], nums[i])`

简化理解：如果前面的累加和是负数，就丢弃；否则继续累加。

### 代码

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        # current_sum: 以当前位置结尾的最大子数组和
        # max_sum: 全局最大子数组和
        current_sum = max_sum = nums[0]

        for num in nums[1:]:
            # 如果前面累加和为负，从当前位置重新开始
            current_sum = max(current_sum + num, num)
            # 更新全局最大值
            max_sum = max(max_sum, current_sum)

        return max_sum
```

## 56. 合并区间 【中等】

### 思路

**排序 + 贪心**

1. 按区间起点排序
2. 遍历区间，判断当前区间与结果中最后一个区间是否重叠
3. 重叠则合并（更新右端点），不重叠则直接加入结果

关键判断：`当前起点 <= 上一个终点` 说明重叠

### 代码

```python
class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        # 按区间起点排序
        intervals.sort(key=lambda x: x[0])

        result = []
        for interval in intervals:
            # 如果结果为空，或当前区间与上一个不重叠
            if not result or interval[0] > result[-1][1]:
                result.append(interval)
            else:
                # 重叠，合并区间（更新右端点）
                result[-1][1] = max(result[-1][1], interval[1])

        return result
```

## 189. 轮转数组 【中等】

### 思路

**三次翻转法**

将数组右移k位等价于：
1. 整体翻转
2. 翻转前k个
3. 翻转后n-k个

例如 `[1,2,3,4,5,6,7]`, k=3:
- 整体翻转: `[7,6,5,4,3,2,1]`
- 前3翻转: `[5,6,7,4,3,2,1]`
- 后4翻转: `[5,6,7,1,2,3,4]`

注意：k可能大于n，需要取模

### 代码

```python
class Solution:
    def rotate(self, nums: List[int], k: int) -> None:
        n = len(nums)
        k = k % n  # 处理k大于数组长度的情况

        def reverse(left: int, right: int) -> None:
            """原地翻转数组的[left, right]部分"""
            while left < right:
                nums[left], nums[right] = nums[right], nums[left]
                left += 1
                right -= 1

        # 三次翻转
        reverse(0, n - 1)      # 整体翻转
        reverse(0, k - 1)      # 翻转前k个
        reverse(k, n - 1)      # 翻转后n-k个
```

## 238. 除自身以外数组的乘积 【中等】

### 思路

**前缀积 + 后缀积**

对于位置i：`answer[i] = 左边所有数的乘积 × 右边所有数的乘积`

优化空间：
1. 第一次遍历：answer存储左侧前缀积
2. 第二次遍历：用一个变量累积右侧后缀积，边遍历边更新answer

### 代码

```python
class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        n = len(nums)
        answer = [1] * n

        # 第一遍：计算每个位置左侧的乘积
        left_product = 1
        for i in range(n):
            answer[i] = left_product
            left_product *= nums[i]

        # 第二遍：计算每个位置右侧的乘积，同时更新答案
        right_product = 1
        for i in range(n - 1, -1, -1):
            answer[i] *= right_product
            right_product *= nums[i]

        return answer
```

## 41. 缺失的第一个正数 【困难】

### 思路

**原地哈希（数组当哈希表）**

核心思想：长度为n的数组，缺失的第一个正数一定在[1, n+1]范围内。

将数组本身当作哈希表：把数字x放到索引x-1的位置。

遍历后，第一个`nums[i] != i+1`的位置i，答案就是i+1。

交换条件：
1. nums[i]在[1, n]范围内
2. nums[i]不在正确位置（避免死循环）

### 代码

```python
class Solution:
    def firstMissingPositive(self, nums: List[int]) -> int:
        n = len(nums)

        # 原地哈希：把数字x放到索引x-1的位置
        for i in range(n):
            # 当前数在有效范围内，且不在正确位置时，交换
            while 1 <= nums[i] <= n and nums[nums[i] - 1] != nums[i]:
                # 把nums[i]放到正确位置nums[i]-1
                correct_idx = nums[i] - 1
                nums[i], nums[correct_idx] = nums[correct_idx], nums[i]

        # 找第一个不在正确位置的数
        for i in range(n):
            if nums[i] != i + 1:
                return i + 1

        # 所有数都在正确位置，返回n+1
        return n + 1
```
