## 160. 相交链表 【简单】

**思路**：双指针法。指针 A 走完链表 A 后走链表 B，指针 B 走完链表 B 后走链表 A。若相交，两指针会在交点相遇（走过相同距离）；若不相交，都会走到 None。

```python
class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        if not headA or not headB:
            return None

        pa, pb = headA, headB
        # 两指针分别走完两条链表，最终会在交点或None相遇
        while pa != pb:
            pa = pa.next if pa else headB
            pb = pb.next if pb else headA

        return pa
```

## 206. 反转链表 【简单】

**思路**：迭代法，用 prev 和 curr 两个指针，逐个反转节点的 next 指向。

```python
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        prev, curr = None, head

        while curr:
            next_temp = curr.next  # 保存下一个节点
            curr.next = prev       # 反转指向
            prev = curr            # prev 前进
            curr = next_temp       # curr 前进

        return prev
```

## 234. 回文链表 【简单】

**思路**：快慢指针找中点 + 反转后半部分 + 比较前后两半。空间 O(1)。

```python
class Solution:
    def isPalindrome(self, head: Optional[ListNode]) -> bool:
        if not head or not head.next:
            return True

        # 1. 快慢指针找中点
        slow, fast = head, head
        while fast.next and fast.next.next:
            slow = slow.next
            fast = fast.next.next

        # 2. 反转后半部分（slow.next 开始）
        second_half = self.reverse(slow.next)

        # 3. 比较前后两半
        p1, p2 = head, second_half
        result = True
        while p2:  # 后半部分可能更短
            if p1.val != p2.val:
                result = False
                break
            p1 = p1.next
            p2 = p2.next

        return result

    def reverse(self, head):
        prev, curr = None, head
        while curr:
            next_temp = curr.next
            curr.next = prev
            prev = curr
            curr = next_temp
        return prev
```

## 141. 环形链表 【简单】

**思路**：快慢指针。快指针每次走2步，慢指针每次走1步。若有环，快慢指针一定会相遇；若无环，快指针先到达 None。

```python
class Solution:
    def hasCycle(self, head: Optional[ListNode]) -> bool:
        if not head or not head.next:
            return False

        slow, fast = head, head

        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
            if slow == fast:  # 相遇说明有环
                return True

        return False
```

## 142. 环形链表 II 【中等】

**思路**：快慢指针相遇后，将一个指针移到头部，两指针同速前进，再次相遇点即为环入口。数学证明：设头到入口距离 a，入口到相遇点 b，相遇点再到入口 c，则 a = c。

```python
class Solution:
    def detectCycle(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head or not head.next:
            return None

        slow, fast = head, head

        # 第一次相遇
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
            if slow == fast:
                break
        else:
            return None  # 无环

        # 一个指针回到头部，同速前进
        slow = head
        while slow != fast:
            slow = slow.next
            fast = fast.next

        return slow
```

## 21. 合并两个有序链表 【简单】

**思路**：用哨兵节点简化操作，比较两链表当前节点，较小者接入结果链表。

```python
class Solution:
    def mergeTwoLists(self, list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(-1)  # 哨兵节点
        curr = dummy

        while list1 and list2:
            if list1.val <= list2.val:
                curr.next = list1
                list1 = list1.next
            else:
                curr.next = list2
                list2 = list2.next
            curr = curr.next

        # 接上剩余部分
        curr.next = list1 if list1 else list2

        return dummy.next
```

## 2. 两数相加 【中等】

**思路**：模拟加法，逐位相加并处理进位。用哨兵节点简化头节点处理。

```python
class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(0)
        curr = dummy
        carry = 0  # 进位

        while l1 or l2 or carry:
            val1 = l1.val if l1 else 0
            val2 = l2.val if l2 else 0

            total = val1 + val2 + carry
            carry = total // 10
            curr.next = ListNode(total % 10)
            curr = curr.next

            if l1: l1 = l1.next
            if l2: l2 = l2.next

        return dummy.next
```

## 19. 删除链表的倒数第 N 个结点 【中等】

**思路**：快慢指针，快指针先走 n 步，然后快慢同时走，当快指针到末尾时，慢指针指向待删除节点的前一个节点。使用哨兵节点处理删除头节点的情况。

```python
class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        fast = slow = dummy

        # 快指针先走 n+1 步
        for _ in range(n + 1):
            fast = fast.next

        # 同时前进，直到 fast 到末尾
        while fast:
            fast = fast.next
            slow = slow.next

        # 删除 slow 的下一个节点
        slow.next = slow.next.next

        return dummy.next
```

## 24. 两两交换链表中的节点 【中等】

**思路**：迭代法，每次处理两个节点的交换。需要记录前一对的尾节点，用于连接交换后的头节点。

```python
class Solution:
    def swapPairs(self, head: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        prev = dummy

        while prev.next and prev.next.next:
            # 定义要交换的两个节点
            first = prev.next
            second = prev.next.next

            # 交换
            first.next = second.next
            second.next = first
            prev.next = second

            # 移动 prev 到下一对的前面
            prev = first

        return dummy.next
```

## 25. K 个一组翻转链表 【困难】

**思路**：分组处理，每 K 个节点为一组进行反转。先检查剩余节点是否足够 K 个，不足则保持原样。用哨兵节点和辅助函数简化代码。

```python
class Solution:
    def reverseKGroup(self, head: Optional[ListNode], k: int) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        prev_group_end = dummy

        while True:
            # 检查是否还有 k 个节点
            kth = self.getKth(prev_group_end, k)
            if not kth:
                break

            next_group_start = kth.next

            # 反转当前组 [prev_group_end.next, kth]
            prev, curr = kth.next, prev_group_end.next
            while curr != next_group_start:
                next_temp = curr.next
                curr.next = prev
                prev = curr
                curr = next_temp

            # 连接：原来的第一个节点变成最后一个
            first_in_group = prev_group_end.next
            prev_group_end.next = kth
            prev_group_end = first_in_group

        return dummy.next

    def getKth(self, start, k):
        """从 start 开始，返回第 k 个节点"""
        curr = start
        for _ in range(k):
            if not curr:
                return None
            curr = curr.next
        return curr
```

## 138. 随机链表的复制 【中等】

**思路**：方法一：哈希表存储原节点到新节点的映射。方法二：交织链表法（空间O(1)）。这里用哈希表法，更直观。

```python
class Solution:
    def copyRandomList(self, head: 'Optional[Node]') -> 'Optional[Node]':
        if not head:
            return None

        # 哈希表：原节点 -> 新节点
        old_to_new = {}

        # 第一遍：创建所有新节点
        curr = head
        while curr:
            old_to_new[curr] = Node(curr.val)
            curr = curr.next

        # 第二遍：连接 next 和 random
        curr = head
        while curr:
            new_node = old_to_new[curr]
            new_node.next = old_to_new.get(curr.next)
            new_node.random = old_to_new.get(curr.random)
            curr = curr.next

        return old_to_new[head]
```

## 148. 排序链表 【中等】

**思路**：归并排序。快慢指针找中点，递归排序左右两半，然后合并。时间 O(nlogn)，空间 O(logn)（递归栈）。

```python
class Solution:
    def sortList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        # 基准情况：空或单节点
        if not head or not head.next:
            return head

        # 快慢指针找中点（slow 停在前半部分的最后一个节点）
        slow, fast = head, head.next
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next

        # 断开链表
        mid = slow.next
        slow.next = None

        # 递归排序左右两半
        left = self.sortList(head)
        right = self.sortList(mid)

        # 合并两个有序链表
        return self.merge(left, right)

    def merge(self, l1, l2):
        dummy = ListNode(0)
        curr = dummy
        while l1 and l2:
            if l1.val <= l2.val:
                curr.next = l1
                l1 = l1.next
            else:
                curr.next = l2
                l2 = l2.next
            curr = curr.next
        curr.next = l1 if l1 else l2
        return dummy.next
```

## 23. 合并 K 个升序链表 【困难】

**思路**：分治。每次把链表两两合并，直到只剩下一条链表。这个写法和 `21. 合并两个有序链表` 是同一套模板，更适合面试直接写。

```python
class Solution:
    def mergeKLists(self, lists: List[Optional[ListNode]]) -> Optional[ListNode]:
        if not lists:
            return None

        while len(lists) > 1:
            merged = []
            for i in range(0, len(lists), 2):
                l1 = lists[i]
                l2 = lists[i + 1] if i + 1 < len(lists) else None
                merged.append(self.mergeTwoLists(l1, l2))
            lists = merged

        return lists[0]

    def mergeTwoLists(self, l1, l2):
        dummy = ListNode(0)
        cur = dummy
        while l1 and l2:
            if l1.val <= l2.val:
                cur.next = l1
                l1 = l1.next
            else:
                cur.next = l2
                l2 = l2.next
            cur = cur.next
        cur.next = l1 or l2
        return dummy.next
```

## 146. LRU 缓存 【中等】

**思路**：哈希表 + 双向链表。哈希表实现 O(1) 查找，双向链表维护访问顺序（最近访问的在头部，最久未访问的在尾部）。

```python
class DLinkedNode:
    def __init__(self, key=0, value=0):
        self.key = key
        self.value = value
        self.prev = None
        self.next = None

class LRUCache:
    def __init__(self, capacity: int):
        self.capacity = capacity
        self.cache = {}  # key -> DLinkedNode
        # 使用伪头部和伪尾部简化边界处理
        self.head = DLinkedNode()
        self.tail = DLinkedNode()
        self.head.next = self.tail
        self.tail.prev = self.head

    def get(self, key: int) -> int:
        if key not in self.cache:
            return -1
        node = self.cache[key]
        self._move_to_head(node)  # 移到头部表示最近访问
        return node.value

    def put(self, key: int, value: int) -> None:
        if key in self.cache:
            node = self.cache[key]
            node.value = value
            self._move_to_head(node)
        else:
            node = DLinkedNode(key, value)
            self.cache[key] = node
            self._add_to_head(node)
            if len(self.cache) > self.capacity:
                # 删除尾部节点（最久未使用）
                removed = self._remove_tail()
                del self.cache[removed.key]

    def _add_to_head(self, node):
        node.prev = self.head
        node.next = self.head.next
        self.head.next.prev = node
        self.head.next = node

    def _remove_node(self, node):
        node.prev.next = node.next
        node.next.prev = node.prev

    def _move_to_head(self, node):
        self._remove_node(node)
        self._add_to_head(node)

    def _remove_tail(self):
        node = self.tail.prev
        self._remove_node(node)
        return node
```
