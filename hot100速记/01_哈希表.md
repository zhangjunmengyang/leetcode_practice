## 1. 两数之和 【简单】

### 思路

**核心思想：哈希表存储已遍历元素，O(1) 查找配对值**

- 遍历数组，对于每个数 `num`，计算其配对值 `target - num`
- 用哈希表记录 `{数值: 索引}`，边遍历边查找
- 若配对值已在哈希表中，直接返回两个索引

**时间复杂度：O(n)，空间复杂度：O(n)**

### 代码

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        # 哈希表：数值 -> 索引
        hash_map = {}

        for i, num in enumerate(nums):
            complement = target - num  # 计算配对值

            # 如果配对值已存在，返回结果
            if complement in hash_map:
                return [hash_map[complement], i]

            # 将当前数存入哈希表
            hash_map[num] = i

        return []
```

## 49. 字母异位词分组 【中等】

### 思路

**核心思想：异位词排序后相同，以排序结果作为哈希表的 key**

- 字母异位词的特点：字符组成相同，顺序不同
- 对每个字符串排序，排序后的结果作为分组依据
- 用哈希表 `{排序后字符串: [原字符串列表]}` 进行分组

**时间复杂度：O(n * k * log k)，其中 k 为字符串最大长度**

### 代码

```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        from collections import defaultdict

        # 哈希表：排序后的字符串 -> 原字符串列表
        groups = defaultdict(list)

        for s in strs:
            # 排序字符串作为 key（tuple 可哈希）
            key = tuple(sorted(s))
            groups[key].append(s)

        return list(groups.values())
```

**替代方案：用字符计数作为 key（避免排序）**

```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        from collections import defaultdict

        groups = defaultdict(list)

        for s in strs:
            # 统计26个字母出现次数作为 key
            count = [0] * 26
            for c in s:
                count[ord(c) - ord('a')] += 1
            groups[tuple(count)].append(s)

        return list(groups.values())
```

## 128. 最长连续序列 【中等】

### 思路

**核心思想：哈希集合 + 只从序列起点开始计数**

- 将数组存入哈希集合，实现 O(1) 查找
- 遍历每个数，**只有当 `num - 1` 不存在时**，才开始计数（确保从序列起点开始）
- 从起点向后连续查找 `num + 1, num + 2, ...`，统计长度

**关键优化：跳过非起点，避免重复计算，保证 O(n) 复杂度**

**时间复杂度：O(n)，空间复杂度：O(n)**

### 代码

```python
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        if not nums:
            return 0

        # 转为集合，O(1) 查找
        num_set = set(nums)
        max_length = 0

        for num in num_set:
            # 关键：只从序列起点开始计数（前一个数不存在）
            if num - 1 not in num_set:
                current_num = num
                current_length = 1

                # 向后连续查找
                while current_num + 1 in num_set:
                    current_num += 1
                    current_length += 1

                max_length = max(max_length, current_length)

        return max_length
```

## 哈希表专题总结

| 题目 | 核心技巧 | 哈希表用途 |
|------|---------|-----------|
| 两数之和 | 边遍历边查找 | 存储 {值: 索引} |
| 字母异位词分组 | 排序/计数作为 key | 分组聚合 |
| 最长连续序列 | 只从起点计数 | O(1) 存在性判断 |

**哈希表核心价值：将 O(n) 查找优化为 O(1)**
