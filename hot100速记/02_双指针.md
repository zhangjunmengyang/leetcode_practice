# LeetCode Hot100 - 双指针专题

## 283. 移动零 【简单】

### 思路
**快慢指针原地交换**
- `slow` 指向下一个非零元素应放置的位置
- `fast` 遍历数组，遇到非零元素就与 `slow` 位置交换
- 核心：保持 `[0, slow)` 区间全是非零元素

### 代码
```python
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        slow = 0  # 指向下一个非零元素应该放置的位置
        for fast in range(len(nums)):
            if nums[fast] != 0:
                # 交换，将非零元素移到前面
                nums[slow], nums[fast] = nums[fast], nums[slow]
                slow += 1
```

## 11. 盛最多水的容器 【中等】

### 思路
**左右双指针 + 贪心**
- 两指针从两端向中间移动
- 面积 = `min(height[l], height[r]) * (r - l)`
- **关键**：每次移动较短的那根柱子（移动长的不可能得到更大面积）

### 代码
```python
class Solution:
    def maxArea(self, height: List[int]) -> int:
        l, r = 0, len(height) - 1
        ans = 0
        while l < r:
            # 计算当前面积，取较短边 * 宽度
            area = min(height[l], height[r]) * (r - l)
            ans = max(ans, area)
            # 移动较短的一边，才可能找到更大面积
            if height[l] < height[r]:
                l += 1
            else:
                r -= 1
        return ans
```

## 15. 三数之和 【中等】

### 思路
**排序 + 固定一个数 + 双指针找两数之和**
- 先排序，方便去重和双指针
- 固定 `nums[i]`，在 `[i+1, n-1]` 范围用双指针找 `target = -nums[i]`
- **去重**：跳过相同的 `i`、`l`、`r`

### 代码
```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        nums.sort()  # 排序是关键
        n = len(nums)
        ans = []

        for i in range(n - 2):
            # 去重：跳过相同的 nums[i]
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            # 剪枝：最小三数之和已经大于0
            if nums[i] + nums[i + 1] + nums[i + 2] > 0:
                break
            # 剪枝：当前数加最大两数仍小于0
            if nums[i] + nums[n - 2] + nums[n - 1] < 0:
                continue

            # 双指针找两数之和等于 -nums[i]
            l, r = i + 1, n - 1
            target = -nums[i]
            while l < r:
                s = nums[l] + nums[r]
                if s == target:
                    ans.append([nums[i], nums[l], nums[r]])
                    l += 1
                    r -= 1
                    # 去重：跳过相同的 l 和 r
                    while l < r and nums[l] == nums[l - 1]:
                        l += 1
                    while l < r and nums[r] == nums[r + 1]:
                        r -= 1
                elif s < target:
                    l += 1
                else:
                    r -= 1

        return ans
```

## 42. 接雨水 【困难】

### 思路
**双指针 + 动态维护左右最大高度**
- 核心公式：位置 `i` 能接的雨水 = `min(左边最高, 右边最高) - height[i]`
- 用 `left_max` 和 `right_max` 分别记录左右两侧遍历过的最大高度
- **关键**：哪边最大值小，就处理哪边（因为接水量由较小的一边决定）

### 代码
```python
class Solution:
    def trap(self, height: List[int]) -> int:
        if not height:
            return 0

        l, r = 0, len(height) - 1
        left_max = right_max = 0  # 记录左右两侧的最大高度
        ans = 0

        while l < r:
            left_max = max(left_max, height[l])
            right_max = max(right_max, height[r])

            # 哪边最大值小，就计算哪边（短板决定接水量）
            if left_max < right_max:
                # 左边是短板，计算左边能接的水
                ans += left_max - height[l]
                l += 1
            else:
                # 右边是短板，计算右边能接的水
                ans += right_max - height[r]
                r -= 1

        return ans
```

## 双指针技巧总结

| 题目 | 指针类型 | 核心思想 |
|------|----------|----------|
| 移动零 | 快慢指针 | 慢指针维护有效区间 |
| 盛水容器 | 左右指针 | 贪心移动短板 |
| 三数之和 | 固定+左右指针 | 排序+去重+双指针 |
| 接雨水 | 左右指针 | 动态维护两侧最值 |
