## 94. 二叉树的中序遍历 【简单】

**思路**：中序遍历顺序为「左-根-右」。递归法直接按顺序访问；迭代法用栈模拟，先一路向左入栈，出栈时访问节点并转向右子树。

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right

class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        res = []
        stack = []
        cur = root

        while cur or stack:
            # 一路向左，全部入栈
            while cur:
                stack.append(cur)
                cur = cur.left
            # 出栈，访问节点
            cur = stack.pop()
            res.append(cur.val)
            # 转向右子树
            cur = cur.right

        return res
```

## 104. 二叉树的最大深度 【简单】

**思路**：递归：当前节点深度 = 1 + max(左子树深度, 右子树深度)。空节点返回0。

```python
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if not root:
            return 0
        # 递归计算左右子树深度，取较大值+1
        return 1 + max(self.maxDepth(root.left), self.maxDepth(root.right))
```

## 226. 翻转二叉树 【简单】

**思路**：递归交换每个节点的左右子树。先递归翻转左右子树，再交换左右子节点。

```python
class Solution:
    def invertTree(self, root: Optional[TreeNode]) -> Optional[TreeNode]:
        if not root:
            return None
        # 递归翻转左右子树
        left = self.invertTree(root.left)
        right = self.invertTree(root.right)
        # 交换左右子节点
        root.left, root.right = right, left
        return root
```

## 101. 对称二叉树 【简单】

**思路**：判断左右子树是否镜像对称。递归比较：左子树的左 vs 右子树的右，左子树的右 vs 右子树的左。

```python
class Solution:
    def isSymmetric(self, root: Optional[TreeNode]) -> bool:
        def isMirror(left: TreeNode, right: TreeNode) -> bool:
            # 都为空，对称
            if not left and not right:
                return True
            # 一个为空一个不为空，不对称
            if not left or not right:
                return False
            # 值相等且子树镜像对称
            return (left.val == right.val and
                    isMirror(left.left, right.right) and
                    isMirror(left.right, right.left))

        return isMirror(root.left, root.right) if root else True
```

## 543. 二叉树的直径 【简单】

**思路**：直径 = 某节点左子树深度 + 右子树深度的最大值。在求深度的过程中更新全局最大直径。

```python
class Solution:
    def diameterOfBinaryTree(self, root: Optional[TreeNode]) -> int:
        self.diameter = 0

        def depth(node: TreeNode) -> int:
            if not node:
                return 0
            left = depth(node.left)
            right = depth(node.right)
            # 更新直径：经过当前节点的路径长度
            self.diameter = max(self.diameter, left + right)
            # 返回当前节点的深度
            return 1 + max(left, right)

        depth(root)
        return self.diameter
```

## 102. 二叉树的层序遍历 【中等】

**思路**：BFS，用队列逐层遍历。每次处理一层，记录当前层节点数，依次出队并将子节点入队。

```python
from collections import deque

class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root:
            return []

        res = []
        queue = deque([root])

        while queue:
            level = []
            size = len(queue)  # 当前层的节点数
            for _ in range(size):
                node = queue.popleft()
                level.append(node.val)
                # 子节点入队
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)
            res.append(level)

        return res
```

## 108. 将有序数组转换为二叉搜索树 【简单】

**思路**：有序数组的中间元素作为根节点，左半部分构建左子树，右半部分构建右子树，递归构建平衡BST。

```python
class Solution:
    def sortedArrayToBST(self, nums: List[int]) -> Optional[TreeNode]:
        def build(left: int, right: int) -> TreeNode:
            if left > right:
                return None
            # 选择中间元素作为根节点
            mid = (left + right) // 2
            node = TreeNode(nums[mid])
            # 递归构建左右子树
            node.left = build(left, mid - 1)
            node.right = build(mid + 1, right)
            return node

        return build(0, len(nums) - 1)
```

## 98. 验证二叉搜索树 【中等】

**思路**：BST的中序遍历是递增序列。可用递归验证每个节点值在(lower, upper)范围内，或中序遍历检查是否严格递增。

```python
class Solution:
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        def validate(node: TreeNode, lower: float, upper: float) -> bool:
            if not node:
                return True
            # 当前节点值必须在(lower, upper)范围内
            if node.val <= lower or node.val >= upper:
                return False
            # 左子树值 < 当前值，右子树值 > 当前值
            return (validate(node.left, lower, node.val) and
                    validate(node.right, node.val, upper))

        return validate(root, float('-inf'), float('inf'))
```

## 230. 二叉搜索树中第 K 小的元素 【中等】

**思路**：BST中序遍历得到升序序列，第K个访问的就是第K小。用迭代法可以提前终止。

```python
class Solution:
    def kthSmallest(self, root: Optional[TreeNode], k: int) -> int:
        stack = []
        cur = root

        while cur or stack:
            # 一路向左
            while cur:
                stack.append(cur)
                cur = cur.left
            cur = stack.pop()
            k -= 1
            # 找到第k小
            if k == 0:
                return cur.val
            cur = cur.right

        return -1
```

## 199. 二叉树的右视图 【中等】

**思路**：层序遍历，每层最后一个节点即为右视图可见节点。或DFS优先访问右子树，记录每层第一个访问的节点。

```python
from collections import deque

class Solution:
    def rightSideView(self, root: Optional[TreeNode]) -> List[int]:
        if not root:
            return []

        res = []
        queue = deque([root])

        while queue:
            size = len(queue)
            for i in range(size):
                node = queue.popleft()
                # 每层最后一个节点加入结果
                if i == size - 1:
                    res.append(node.val)
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)

        return res
```

## 114. 二叉树展开为链表 【中等】

**思路**：将左子树插入到右子树的位置，原右子树接到左子树最右节点。按前序遍历顺序依次处理每个节点。

```python
class Solution:
    def flatten(self, root: Optional[TreeNode]) -> None:
        """
        Do not return anything, modify root in-place instead.
        """
        cur = root
        while cur:
            if cur.left:
                # 找到左子树的最右节点
                pre = cur.left
                while pre.right:
                    pre = pre.right
                # 将右子树接到左子树最右节点
                pre.right = cur.right
                # 左子树移到右边
                cur.right = cur.left
                cur.left = None
            # 处理下一个节点
            cur = cur.right
```

## 105. 从前序与中序遍历序列构造二叉树 【中等】

**思路**：前序第一个元素是根，在中序中找到根的位置，左边是左子树，右边是右子树，递归构建。用哈希表加速查找。

```python
class Solution:
    def buildTree(self, preorder: List[int], inorder: List[int]) -> Optional[TreeNode]:
        # 哈希表存储中序遍历的索引
        idx_map = {val: idx for idx, val in enumerate(inorder)}

        def build(pre_left: int, pre_right: int, in_left: int, in_right: int) -> TreeNode:
            if pre_left > pre_right:
                return None

            # 前序遍历第一个元素是根节点
            root_val = preorder[pre_left]
            root = TreeNode(root_val)

            # 在中序遍历中找到根节点位置
            in_root = idx_map[root_val]
            # 左子树的节点数
            left_size = in_root - in_left

            # 递归构建左右子树
            root.left = build(pre_left + 1, pre_left + left_size, in_left, in_root - 1)
            root.right = build(pre_left + left_size + 1, pre_right, in_root + 1, in_right)

            return root

        return build(0, len(preorder) - 1, 0, len(inorder) - 1)
```

## 437. 路径总和 III 【中等】

**思路**：前缀和 + 哈希表。记录从根到当前节点的前缀和，路径和为targetSum等价于存在前缀和差为targetSum。

```python
from collections import defaultdict

class Solution:
    def pathSum(self, root: Optional[TreeNode], targetSum: int) -> int:
        # 前缀和计数，初始化0出现1次
        prefix_count = defaultdict(int)
        prefix_count[0] = 1

        def dfs(node: TreeNode, curr_sum: int) -> int:
            if not node:
                return 0

            # 当前前缀和
            curr_sum += node.val
            # 找是否存在前缀和为 curr_sum - targetSum
            count = prefix_count[curr_sum - targetSum]

            # 记录当前前缀和
            prefix_count[curr_sum] += 1
            # 递归左右子树
            count += dfs(node.left, curr_sum)
            count += dfs(node.right, curr_sum)
            # 回溯，移除当前前缀和
            prefix_count[curr_sum] -= 1

            return count

        return dfs(root, 0)
```

## 236. 二叉树的最近公共祖先 【中等】

**思路**：递归查找p和q。若当前节点为p或q则返回当前节点；若p和q分别在左右子树，则当前节点为LCA；否则返回非空的那个子树结果。

```python
class Solution:
    def lowestCommonAncestor(self, root: 'TreeNode', p: 'TreeNode', q: 'TreeNode') -> 'TreeNode':
        # 递归终止条件
        if not root or root == p or root == q:
            return root

        # 递归查找左右子树
        left = self.lowestCommonAncestor(root.left, p, q)
        right = self.lowestCommonAncestor(root.right, p, q)

        # p和q分别在左右子树，当前节点就是LCA
        if left and right:
            return root
        # 都在左子树或都在右子树
        return left if left else right
```

## 124. 二叉树中的最大路径和 【困难】

**思路**：对每个节点计算「经过该节点的最大路径和」，更新全局最大值。递归返回「以该节点为端点的最大贡献值」（只能选左或右分支）。

```python
class Solution:
    def maxPathSum(self, root: Optional[TreeNode]) -> int:
        self.max_sum = float('-inf')

        def maxGain(node: TreeNode) -> int:
            if not node:
                return 0

            # 递归计算左右子树的最大贡献值（负数则取0，表示不选）
            left_gain = max(maxGain(node.left), 0)
            right_gain = max(maxGain(node.right), 0)

            # 经过当前节点的最大路径和
            path_sum = node.val + left_gain + right_gain
            # 更新全局最大值
            self.max_sum = max(self.max_sum, path_sum)

            # 返回以当前节点为端点的最大贡献值（只能选一边）
            return node.val + max(left_gain, right_gain)

        maxGain(root)
        return self.max_sum
```
