## 200. 岛屿数量 【中等】

### 思路

**核心算法：DFS/BFS 连通分量计数**

1. 遍历整个网格，遇到陆地 `'1'` 时，岛屿数量 +1
2. 从该位置启动 DFS/BFS，将所有相连的陆地标记为已访问（改为 `'0'`）
3. 继续遍历，直到遍历完整个网格

**关键点：**
- 原地修改避免额外空间（将访问过的 `'1'` 改为 `'0'`）
- 四个方向扩展：上、下、左、右

### 代码

```python
class Solution:
    def numIslands(self, grid: List[List[str]]) -> int:
        if not grid:
            return 0

        m, n = len(grid), len(grid[0])
        count = 0

        def dfs(i, j):
            # 边界检查 或 遇到水
            if i < 0 or i >= m or j < 0 or j >= n or grid[i][j] == '0':
                return
            # 标记为已访问
            grid[i][j] = '0'
            # 向四个方向扩展
            dfs(i + 1, j)
            dfs(i - 1, j)
            dfs(i, j + 1)
            dfs(i, j - 1)

        for i in range(m):
            for j in range(n):
                if grid[i][j] == '1':
                    count += 1  # 发现新岛屿
                    dfs(i, j)   # 淹没整个岛屿

        return count
```

## 994. 腐烂的橘子 【中等】

### 思路

**核心算法：多源 BFS**

1. 统计新鲜橘子数量，将所有腐烂橘子入队（多源起点）
2. BFS 层序遍历，每一层代表过了 1 分钟
3. 腐烂橘子感染相邻的新鲜橘子，新鲜橘子数量减少
4. 遍历完成后，若还有新鲜橘子，返回 -1；否则返回经过的时间

**关键点：**
- 多源 BFS：所有腐烂橘子同时开始扩散
- 按层遍历记录时间（分钟数）

### 代码

```python
class Solution:
    def orangesRotting(self, grid: List[List[int]]) -> int:
        from collections import deque

        m, n = len(grid), len(grid[0])
        queue = deque()
        fresh = 0

        # 统计新鲜橘子，收集腐烂橘子位置
        for i in range(m):
            for j in range(n):
                if grid[i][j] == 2:
                    queue.append((i, j))
                elif grid[i][j] == 1:
                    fresh += 1

        # 没有新鲜橘子，直接返回 0
        if fresh == 0:
            return 0

        minutes = 0
        directions = [(0, 1), (0, -1), (1, 0), (-1, 0)]

        # BFS 层序遍历
        while queue:
            minutes += 1
            for _ in range(len(queue)):  # 当前层的所有腐烂橘子
                x, y = queue.popleft()
                for dx, dy in directions:
                    nx, ny = x + dx, y + dy
                    # 感染相邻的新鲜橘子
                    if 0 <= nx < m and 0 <= ny < n and grid[nx][ny] == 1:
                        grid[nx][ny] = 2
                        fresh -= 1
                        queue.append((nx, ny))

        # 减去多计算的一轮（最后一轮没有新感染）
        return minutes - 1 if fresh == 0 else -1
```

## 207. 课程表 【中等】

### 思路

**核心算法：拓扑排序（BFS / Kahn 算法）**

1. 建立邻接表和入度数组
2. 将所有入度为 0 的节点入队（无前置课程）
3. BFS 遍历：弹出节点，将其指向的节点入度 -1
4. 若入度减为 0，则入队
5. 最终检查是否所有课程都被处理（有环则无法完成）

**关键点：**
- 有向图判环问题 = 拓扑排序
- 能完成拓扑排序（处理 n 个节点）= 无环 = 能修完所有课

### 代码

```python
class Solution:
    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:
        from collections import deque

        # 建图：邻接表 + 入度数组
        indegree = [0] * numCourses
        adj = [[] for _ in range(numCourses)]

        for cur, pre in prerequisites:
            adj[pre].append(cur)  # pre -> cur
            indegree[cur] += 1

        # 入度为 0 的节点入队
        queue = deque([i for i in range(numCourses) if indegree[i] == 0])
        count = 0  # 已处理的课程数

        while queue:
            course = queue.popleft()
            count += 1
            # 将该课程指向的课程入度 -1
            for next_course in adj[course]:
                indegree[next_course] -= 1
                if indegree[next_course] == 0:
                    queue.append(next_course)

        # 能处理所有课程 = 无环
        return count == numCourses
```

## 208. 实现 Trie (前缀树) 【中等】

### 思路

**核心算法：前缀树（字典树）**

1. 每个节点包含：子节点映射（字典）+ 是否为单词结尾标记
2. `insert`：逐字符创建/遍历节点，最后标记结束
3. `search`：逐字符遍历，检查是否存在且是单词结尾
4. `startsWith`：逐字符遍历，只检查前缀是否存在

**关键点：**
- 用字典存储子节点，key 为字符，value 为子节点
- `is_end` 标记区分完整单词和前缀

### 代码

```python
class Trie:

    def __init__(self):
        self.children = {}  # 子节点映射
        self.is_end = False  # 是否为单词结尾

    def insert(self, word: str) -> None:
        node = self
        for char in word:
            if char not in node.children:
                node.children[char] = Trie()  # 创建新节点
            node = node.children[char]
        node.is_end = True  # 标记单词结束

    def search(self, word: str) -> bool:
        node = self._search_prefix(word)
        return node is not None and node.is_end

    def startsWith(self, prefix: str) -> bool:
        return self._search_prefix(prefix) is not None

    def _search_prefix(self, prefix: str):
        """查找前缀对应的节点"""
        node = self
        for char in prefix:
            if char not in node.children:
                return None
            node = node.children[char]
        return node
```
