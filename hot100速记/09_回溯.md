# LeetCode Hot100 回溯专题速记

> 回溯算法核心模板：
> 1. 做选择 -> 2. 递归 -> 3. 撤销选择
> 关键：理解什么时候收集结果，什么时候剪枝

---

## 46. 全排列 【中等】

### 思路
- 经典回溯模板题，每次从未使用的数字中选一个加入路径
- 使用 `used` 数组标记已使用的数字
- 当路径长度等于数组长度时，收集结果

### 代码
```python
class Solution:
    def permute(self, nums: List[int]) -> List[List[int]]:
        res = []
        path = []
        used = [False] * len(nums)

        def backtrack():
            # 收集结果：路径长度等于数组长度
            if len(path) == len(nums):
                res.append(path[:])  # 注意要拷贝
                return

            for i in range(len(nums)):
                if used[i]:  # 跳过已使用的数字
                    continue
                # 做选择
                path.append(nums[i])
                used[i] = True
                # 递归
                backtrack()
                # 撤销选择
                path.pop()
                used[i] = False

        backtrack()
        return res
```

## 78. 子集 【中等】

### 思路
- 子集问题：每个元素都有「选」或「不选」两种状态
- 方法一：每层递归都是一个有效子集，直接收集
- 方法二：用 start 控制遍历起点，避免重复

### 代码
```python
class Solution:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        res = []
        path = []

        def backtrack(start):
            # 每个节点都是一个子集，直接收集
            res.append(path[:])

            for i in range(start, len(nums)):
                path.append(nums[i])  # 做选择
                backtrack(i + 1)       # 递归，注意是 i+1 不是 start+1
                path.pop()             # 撤销选择

        backtrack(0)
        return res
```

## 17. 电话号码的字母组合 【中等】

### 思路
- 多个集合的组合问题
- 用哈希表存储数字到字母的映射
- 每层递归处理一个数字，遍历该数字对应的所有字母

### 代码
```python
class Solution:
    def letterCombinations(self, digits: str) -> List[str]:
        if not digits:
            return []

        # 数字到字母的映射
        phone = {
            '2': 'abc', '3': 'def', '4': 'ghi', '5': 'jkl',
            '6': 'mno', '7': 'pqrs', '8': 'tuv', '9': 'wxyz'
        }

        res = []

        def backtrack(index, path):
            # 收集结果：处理完所有数字
            if index == len(digits):
                res.append(''.join(path))
                return

            # 获取当前数字对应的字母
            letters = phone[digits[index]]
            for letter in letters:
                path.append(letter)      # 做选择
                backtrack(index + 1, path)  # 递归下一个数字
                path.pop()               # 撤销选择

        backtrack(0, [])
        return res
```

## 39. 组合总和 【中等】

### 思路
- 同一个数字可以无限制重复选取
- 用 start 避免重复组合（如 [2,2,3] 和 [2,3,2]）
- 剪枝：当前和超过 target 时直接返回

### 代码
```python
class Solution:
    def combinationSum(self, candidates: List[int], target: int) -> List[List[int]]:
        res = []
        path = []

        def backtrack(start, remain):
            # 找到目标
            if remain == 0:
                res.append(path[:])
                return
            # 剪枝：超过目标
            if remain < 0:
                return

            for i in range(start, len(candidates)):
                path.append(candidates[i])
                # 注意这里是 i 而不是 i+1，因为可以重复使用
                backtrack(i, remain - candidates[i])
                path.pop()

        backtrack(0, target)
        return res
```

## 22. 括号生成 【中等】

### 思路
- 左括号数量 < n 时可以加左括号
- 右括号数量 < 左括号数量时可以加右括号
- 当左右括号都用完时收集结果

### 代码
```python
class Solution:
    def generateParenthesis(self, n: int) -> List[str]:
        res = []

        def backtrack(path, left, right):
            # 收集结果：左右括号都用完
            if left == n and right == n:
                res.append(''.join(path))
                return

            # 可以加左括号
            if left < n:
                path.append('(')
                backtrack(path, left + 1, right)
                path.pop()

            # 可以加右括号（右括号数量必须小于左括号）
            if right < left:
                path.append(')')
                backtrack(path, left, right + 1)
                path.pop()

        backtrack([], 0, 0)
        return res
```

## 79. 单词搜索 【中等】

### 思路
- 二维网格上的回溯搜索
- 从每个位置开始尝试匹配单词
- 用临时标记避免重复访问（原地修改或 visited 数组）
- 四个方向 DFS 搜索

### 代码
```python
class Solution:
    def exist(self, board: List[List[str]], word: str) -> bool:
        m, n = len(board), len(board[0])

        def backtrack(i, j, k):
            # 越界或字符不匹配
            if i < 0 or i >= m or j < 0 or j >= n or board[i][j] != word[k]:
                return False
            # 已匹配到最后一个字符
            if k == len(word) - 1:
                return True

            # 标记已访问（原地修改，节省空间）
            temp = board[i][j]
            board[i][j] = '#'

            # 四个方向搜索
            found = (backtrack(i + 1, j, k + 1) or
                     backtrack(i - 1, j, k + 1) or
                     backtrack(i, j + 1, k + 1) or
                     backtrack(i, j - 1, k + 1))

            # 撤销标记
            board[i][j] = temp

            return found

        # 从每个位置开始尝试
        for i in range(m):
            for j in range(n):
                if backtrack(i, j, 0):
                    return True
        return False
```

## 131. 分割回文串 【中等】

### 思路
- 切割问题本质也是组合问题
- 用 start 表示切割起点，枚举切割终点
- 只有当前子串是回文时才继续递归
- 切到字符串末尾时收集结果

### 代码
```python
class Solution:
    def partition(self, s: str) -> List[List[str]]:
        res = []
        path = []

        def is_palindrome(sub):
            return sub == sub[::-1]

        def backtrack(start):
            # 切到末尾，收集结果
            if start == len(s):
                res.append(path[:])
                return

            for end in range(start + 1, len(s) + 1):
                sub = s[start:end]
                # 剪枝：只有回文才继续
                if is_palindrome(sub):
                    path.append(sub)
                    backtrack(end)  # 从 end 开始继续切割
                    path.pop()

        backtrack(0)
        return res
```

## 51. N 皇后 【困难】

### 思路
- 逐行放置皇后，每行只能放一个
- 检查列冲突、主对角线冲突（行-列相同）、副对角线冲突（行+列相同）
- 用三个集合分别记录被占用的列、主对角线、副对角线

### 代码
```python
class Solution:
    def solveNQueens(self, n: int) -> List[List[str]]:
        res = []
        board = [['.'] * n for _ in range(n)]
        cols = set()       # 被占用的列
        diag1 = set()      # 被占用的主对角线 (row - col)
        diag2 = set()      # 被占用的副对角线 (row + col)

        def backtrack(row):
            # 所有行都放置完成
            if row == n:
                res.append([''.join(r) for r in board])
                return

            for col in range(n):
                # 检查是否可以放置
                if col in cols or (row - col) in diag1 or (row + col) in diag2:
                    continue

                # 放置皇后
                board[row][col] = 'Q'
                cols.add(col)
                diag1.add(row - col)
                diag2.add(row + col)

                # 递归下一行
                backtrack(row + 1)

                # 撤销
                board[row][col] = '.'
                cols.remove(col)
                diag1.remove(row - col)
                diag2.remove(row + col)

        backtrack(0)
        return res
```

## 回溯总结

| 题型 | 关键点 | 代表题目 |
|------|--------|----------|
| 排列 | used数组标记 | 46. 全排列 |
| 子集 | start控制起点，每个节点都收集 | 78. 子集 |
| 组合 | start控制起点，达到条件收集 | 39. 组合总和 |
| 切割 | start表示切割位置 | 131. 分割回文串 |
| 棋盘 | 逐行/逐格搜索 | 51. N皇后, 79. 单词搜索 |

**回溯模板**：
```python
def backtrack(路径, 选择列表):
    if 满足结束条件:
        收集结果
        return

    for 选择 in 选择列表:
        if 不合法:
            continue  # 剪枝
        做选择
        backtrack(新路径, 新选择列表)
        撤销选择
```
