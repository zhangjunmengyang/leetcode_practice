## 70. 爬楼梯 【简单】

**【思路】**
- 经典斐波那契问题：到达第n阶的方法 = 到达第n-1阶的方法 + 到达第n-2阶的方法
- 状态转移：`dp[i] = dp[i-1] + dp[i-2]`
- 空间优化：只需保存前两个状态

**【代码】**
```python
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2:
            return n
        # 空间优化：只保存前两个状态
        prev, curr = 1, 2
        for i in range(3, n + 1):
            prev, curr = curr, prev + curr
        return curr
```

## 118. 杨辉三角 【简单】

**【思路】**
- 每个位置的值 = 上一行左上角 + 上一行正上方
- 边界：每行首尾都是1
- 状态转移：`dp[i][j] = dp[i-1][j-1] + dp[i-1][j]`

**【代码】**
```python
class Solution:
    def generate(self, numRows: int) -> List[List[int]]:
        result = []
        for i in range(numRows):
            row = [1] * (i + 1)  # 初始化当前行，全为1
            # 填充中间元素
            for j in range(1, i):
                row[j] = result[i-1][j-1] + result[i-1][j]
            result.append(row)
        return result
```

## 198. 打家劫舍 【中等】

**【思路】**
- 不能偷相邻房屋，对于第i间房屋：偷或不偷
- 状态转移：`dp[i] = max(dp[i-1], dp[i-2] + nums[i])`
- dp[i]表示偷到第i间房屋时的最大金额

**【代码】**
```python
class Solution:
    def rob(self, nums: List[int]) -> int:
        if len(nums) <= 2:
            return max(nums)
        # 空间优化
        prev2, prev1 = nums[0], max(nums[0], nums[1])
        for i in range(2, len(nums)):
            prev2, prev1 = prev1, max(prev1, prev2 + nums[i])
        return prev1
```

## 279. 完全平方数 【中等】

**【思路】**
- 完全背包问题：物品是完全平方数1,4,9,16...，求凑成n的最少物品数
- 状态转移：`dp[i] = min(dp[i], dp[i - j*j] + 1)`，其中j*j <= i
- dp[i]表示凑成i所需的最少完全平方数个数

**【代码】**
```python
class Solution:
    def numSquares(self, n: int) -> int:
        dp = [float('inf')] * (n + 1)
        dp[0] = 0  # 凑成0需要0个数
        for i in range(1, n + 1):
            j = 1
            while j * j <= i:
                dp[i] = min(dp[i], dp[i - j * j] + 1)
                j += 1
        return dp[n]
```

## 322. 零钱兑换 【中等】

**【思路】**
- 完全背包问题：每种硬币可用无限次，求凑成amount的最少硬币数
- 状态转移：`dp[i] = min(dp[i], dp[i - coin] + 1)`
- dp[i]表示凑成金额i所需的最少硬币数

**【代码】**
```python
class Solution:
    def coinChange(self, coins: List[int], amount: int) -> int:
        dp = [float('inf')] * (amount + 1)
        dp[0] = 0  # 凑成0元需要0枚硬币
        for i in range(1, amount + 1):
            for coin in coins:
                if coin <= i and dp[i - coin] != float('inf'):
                    dp[i] = min(dp[i], dp[i - coin] + 1)
        return dp[amount] if dp[amount] != float('inf') else -1
```

## 139. 单词拆分 【中等】

**【思路】**
- dp[i]表示s[0:i]是否可以被拆分成字典中的单词
- 遍历所有可能的分割点j，检查s[0:j]可拆分且s[j:i]在字典中
- 状态转移：`dp[i] = dp[j] and s[j:i] in wordSet`

**【代码】**
```python
class Solution:
    def wordBreak(self, s: str, wordDict: List[str]) -> bool:
        wordSet = set(wordDict)
        n = len(s)
        dp = [False] * (n + 1)
        dp[0] = True  # 空字符串可拆分

        for i in range(1, n + 1):
            for j in range(i):
                # s[0:j]可拆分 且 s[j:i]是字典中的单词
                if dp[j] and s[j:i] in wordSet:
                    dp[i] = True
                    break
        return dp[n]
```

## 300. 最长递增子序列 【中等】

**【思路】**
- dp[i]表示以nums[i]结尾的最长递增子序列长度
- 对于每个i，遍历所有j<i，若nums[j]<nums[i]则可接上
- 状态转移：`dp[i] = max(dp[j] + 1)`，其中j<i且nums[j]<nums[i]
- 这题面试默认先写 O(n^2) DP，最稳最好讲；如果面试官追求优化，再补二分版本

**【代码】**
```python
class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        n = len(nums)
        dp = [1] * n  # 每个元素自身构成长度为1的子序列

        for i in range(1, n):
            for j in range(i):
                if nums[j] < nums[i]:
                    dp[i] = max(dp[i], dp[j] + 1)

        return max(dp)
```

**【进阶：二分优化版】**
```python
class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        # tails[i]表示长度为i+1的递增子序列的最小末尾元素
        tails = []
        for num in nums:
            # 二分查找num应该插入的位置
            left, right = 0, len(tails)
            while left < right:
                mid = (left + right) // 2
                if tails[mid] < num:
                    left = mid + 1
                else:
                    right = mid
            if left == len(tails):
                tails.append(num)
            else:
                tails[left] = num
        return len(tails)
```

## 152. 乘积最大子数组 【中等】

**【思路】**
- 因为负数乘负数会变大，需要同时维护最大值和最小值
- 如果当前数是负数，最大值和最小值的角色会交换
- 每次更新:
  - `cur_max = max(num, cur_max * num)`
  - `cur_min = min(num, cur_min * num)`

**【代码】**
```python
class Solution:
    def maxProduct(self, nums: List[int]) -> int:
        cur_max = cur_min = ans = nums[0]

        for num in nums[1:]:
            if num < 0:
                cur_max, cur_min = cur_min, cur_max
            cur_max = max(num, cur_max * num)
            cur_min = min(num, cur_min * num)
            ans = max(ans, cur_max)

        return ans
```

## 416. 分割等和子集 【中等】

**【思路】**
- 转化为0-1背包：能否从数组中选出若干数，使其和等于总和的一半
- 总和为奇数直接返回False
- dp[j]表示是否能凑出和为j
- 状态转移：`dp[j] = dp[j] or dp[j - nums[i]]`

**【代码】**
```python
class Solution:
    def canPartition(self, nums: List[int]) -> bool:
        total = sum(nums)
        if total % 2 == 1:  # 奇数无法平分
            return False

        target = total // 2
        dp = [False] * (target + 1)
        dp[0] = True  # 和为0总是可以达到

        for num in nums:
            # 倒序遍历，确保每个数只用一次（0-1背包）
            for j in range(target, num - 1, -1):
                dp[j] = dp[j] or dp[j - num]

        return dp[target]
```

## 32. 最长有效括号 【困难】

**【思路】**
- 用栈存下标，栈底先放 `-1` 作为哨兵
- 遇到 `'('` 就入栈
- 遇到 `')'` 就弹栈
- 如果弹完栈空了，说明当前位置不能作为结尾，当前下标入栈当新边界
- 如果栈不空，当前有效长度就是 `i - stack[-1]`

**【代码】**
```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        stack = [-1]
        ans = 0

        for i, ch in enumerate(s):
            if ch == '(':
                stack.append(i)
            else:
                stack.pop()
                if not stack:
                    stack.append(i)
                else:
                    ans = max(ans, i - stack[-1])

        return ans
```

## 62. 不同路径 【中等】

**【思路】**
- 只能向右或向下走，到达(i,j)的路径数 = 从上方来 + 从左方来
- 状态转移：`dp[i][j] = dp[i-1][j] + dp[i][j-1]`
- 边界：第一行和第一列都只有1条路径

**【代码】**
```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        # 空间优化：只用一维数组
        dp = [1] * n  # 第一行全为1

        for i in range(1, m):
            for j in range(1, n):
                dp[j] = dp[j] + dp[j-1]  # 上方 + 左方

        return dp[n-1]
```

## 64. 最小路径和 【中等】

**【思路】**
- 到达(i,j)的最小路径和 = min(从上方来, 从左方来) + 当前格子值
- 状态转移：`dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j]`
- 边界：第一行只能从左来，第一列只能从上来

**【代码】**
```python
class Solution:
    def minPathSum(self, grid: List[List[int]]) -> int:
        m, n = len(grid), len(grid[0])
        # 空间优化：使用一维数组
        dp = [0] * n

        for i in range(m):
            for j in range(n):
                if i == 0 and j == 0:
                    dp[j] = grid[0][0]
                elif i == 0:
                    dp[j] = dp[j-1] + grid[i][j]  # 第一行只能从左来
                elif j == 0:
                    dp[j] = dp[j] + grid[i][j]    # 第一列只能从上来
                else:
                    dp[j] = min(dp[j], dp[j-1]) + grid[i][j]

        return dp[n-1]
```

## 5. 最长回文子串 【中等】

**【思路】**
- 方法1：DP，dp[i][j]表示s[i:j+1]是否为回文
- 方法2：中心扩展法，枚举每个中心点向两边扩展
- 状态转移：`dp[i][j] = (s[i] == s[j]) and dp[i+1][j-1]`

**【代码】**
```python
class Solution:
    def longestPalindrome(self, s: str) -> str:
        n = len(s)
        if n < 2:
            return s

        start, maxLen = 0, 1

        def expandAroundCenter(left: int, right: int) -> int:
            """从中心向两边扩展，返回回文长度"""
            while left >= 0 and right < n and s[left] == s[right]:
                left -= 1
                right += 1
            return right - left - 1

        for i in range(n):
            len1 = expandAroundCenter(i, i)      # 奇数长度回文
            len2 = expandAroundCenter(i, i + 1)  # 偶数长度回文
            curLen = max(len1, len2)
            if curLen > maxLen:
                maxLen = curLen
                start = i - (curLen - 1) // 2

        return s[start:start + maxLen]
```

## 1143. 最长公共子序列 【中等】

**【思路】**
- dp[i][j]表示text1[0:i]和text2[0:j]的最长公共子序列长度
- 若text1[i-1] == text2[j-1]：`dp[i][j] = dp[i-1][j-1] + 1`
- 否则：`dp[i][j] = max(dp[i-1][j], dp[i][j-1])`

**【代码】**
```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        m, n = len(text1), len(text2)
        # dp[i][j]表示text1前i个字符和text2前j个字符的LCS长度
        dp = [[0] * (n + 1) for _ in range(m + 1)]

        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if text1[i-1] == text2[j-1]:
                    dp[i][j] = dp[i-1][j-1] + 1
                else:
                    dp[i][j] = max(dp[i-1][j], dp[i][j-1])

        return dp[m][n]
```

## 72. 编辑距离 【中等】

**【思路】**
- dp[i][j]表示word1[0:i]转换为word2[0:j]所需的最少操作数
- 三种操作：插入、删除、替换
- 状态转移：
  - 若word1[i-1] == word2[j-1]：`dp[i][j] = dp[i-1][j-1]`
  - 否则：`dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1`
    - dp[i-1][j] + 1：删除word1[i-1]
    - dp[i][j-1] + 1：插入word2[j-1]
    - dp[i-1][j-1] + 1：替换

**【代码】**
```python
class Solution:
    def minDistance(self, word1: str, word2: str) -> int:
        m, n = len(word1), len(word2)
        # dp[i][j]表示word1前i个字符转换为word2前j个字符的最少操作数
        dp = [[0] * (n + 1) for _ in range(m + 1)]

        # 初始化边界
        for i in range(m + 1):
            dp[i][0] = i  # word1删除i个字符变成空串
        for j in range(n + 1):
            dp[0][j] = j  # 空串插入j个字符变成word2

        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if word1[i-1] == word2[j-1]:
                    dp[i][j] = dp[i-1][j-1]  # 字符相同，无需操作
                else:
                    dp[i][j] = min(
                        dp[i-1][j] + 1,    # 删除
                        dp[i][j-1] + 1,    # 插入
                        dp[i-1][j-1] + 1   # 替换
                    )

        return dp[m][n]
```

## 动态规划解题模板总结

### 核心步骤
1. **定义状态**：明确dp数组的含义
2. **初始化**：确定边界条件
3. **状态转移**：找出递推关系
4. **遍历顺序**：根据依赖关系确定遍历方向
5. **返回结果**：通常是dp数组的最后一个元素或最大/最小值

### 常见类型
| 类型 | 典型题目 | 特点 |
|------|----------|------|
| 线性DP | 爬楼梯、打家劫舍 | 一维状态，依赖前几个状态 |
| 背包DP | 零钱兑换、分割等和子集 | 物品+容量的选择问题 |
| 区间DP | 最长回文子串、最长有效括号 | 区间合并或分割 |
| 二维DP | 不同路径、最小路径和 | 网格型问题 |
| 序列DP | LIS、LCS、编辑距离 | 两个序列的匹配问题 |

### 空间优化技巧
- 一维优化：当前状态只依赖上一行时，可用一维数组
- 滚动数组：交替使用两行
- 变量优化：只依赖有限个状态时，用变量代替数组
