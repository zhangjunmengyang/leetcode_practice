## 136. 只出现一次的数字 【简单】

**思路：**
- 核心：异或运算的性质
- `a ^ a = 0`（相同数异或为0）
- `a ^ 0 = a`（任何数与0异或为自身）
- 将所有数异或，成对的数会抵消为0，最后剩下的就是只出现一次的数

**代码：**
```python
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        result = 0
        for num in nums:
            result ^= num  # 异或：相同为0，不同为1
        return result
```

## 169. 多数元素 【简单】

**思路：**
- 核心：Boyer-Moore 投票算法
- 维护一个候选人和票数计数器
- 遇到相同的数，票数+1；遇到不同的数，票数-1
- 票数为0时，更换候选人
- 多数元素（出现次数 > n/2）最终一定会胜出

**代码：**
```python
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        candidate = None
        count = 0

        for num in nums:
            if count == 0:
                candidate = num  # 票数归零，更换候选人
            count += 1 if num == candidate else -1  # 同则+1，异则-1

        return candidate
```

## 75. 颜色分类 【中等】

**思路：**
- 核心：荷兰国旗问题，三指针原地排序
- `p0` 指向0的右边界，`p2` 指向2的左边界
- `curr` 遍历数组：
  - 遇到0：与 `p0` 交换，`p0++`，`curr++`
  - 遇到2：与 `p2` 交换，`p2--`（curr不动，因为交换来的数还需判断）
  - 遇到1：`curr++`

**代码：**
```python
class Solution:
    def sortColors(self, nums: List[int]) -> None:
        p0, curr, p2 = 0, 0, len(nums) - 1

        while curr <= p2:
            if nums[curr] == 0:
                nums[p0], nums[curr] = nums[curr], nums[p0]
                p0 += 1
                curr += 1  # p0左边都是0，交换来的一定是1，可以前进
            elif nums[curr] == 2:
                nums[curr], nums[p2] = nums[p2], nums[curr]
                p2 -= 1    # 交换来的数未知，curr不动继续判断
            else:
                curr += 1  # 是1，直接跳过
```

## 31. 下一个排列 【中等】

**思路：**
- 目标：找到下一个字典序更大的排列
- 步骤：
  1. 从右往左找第一个下降的位置 `i`（`nums[i] < nums[i+1]`）
  2. 从右往左找第一个大于 `nums[i]` 的数，与 `nums[i]` 交换
  3. 将 `i` 后面的部分反转（从降序变升序，得到最小排列）
- 若找不到下降位置，说明是最大排列，直接整体反转

**代码：**
```python
class Solution:
    def nextPermutation(self, nums: List[int]) -> None:
        n = len(nums)
        i = n - 2

        # 1. 从右往左找第一个下降点
        while i >= 0 and nums[i] >= nums[i + 1]:
            i -= 1

        if i >= 0:
            # 2. 从右往左找第一个大于nums[i]的数并交换
            j = n - 1
            while nums[j] <= nums[i]:
                j -= 1
            nums[i], nums[j] = nums[j], nums[i]

        # 3. 反转i后面的部分
        left, right = i + 1, n - 1
        while left < right:
            nums[left], nums[right] = nums[right], nums[left]
            left += 1
            right -= 1
```

## 287. 寻找重复数 【中等】

**思路：**
- 核心：快慢指针（Floyd 判圈算法）
- 把数组看成链表：`index -> nums[index]`
- 有重复数意味着有多个位置指向同一个点，形成环
- 步骤：
  1. 快慢指针找相遇点（确认有环）
  2. 一个指针回到起点，两指针同速前进，再次相遇点就是环入口（重复数）

**代码：**
```python
class Solution:
    def findDuplicate(self, nums: List[int]) -> int:
        # 1. 快慢指针找相遇点
        slow = fast = 0
        while True:
            slow = nums[slow]         # 走一步
            fast = nums[nums[fast]]   # 走两步
            if slow == fast:
                break

        # 2. 找环入口（重复数）
        slow = 0
        while slow != fast:
            slow = nums[slow]
            fast = nums[fast]

        return slow
```
