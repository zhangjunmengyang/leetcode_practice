## 3. 无重复字符的最长子串 【中等】

### 思路

**滑动窗口 + 哈希表**

- 维护一个窗口 `[left, right]`，保证窗口内无重复字符
- 用哈希表记录每个字符最后出现的位置
- 遇到重复字符时，将 `left` 跳到重复字符的下一个位置
- 每次更新最大长度 `right - left + 1`

**核心**: 右指针扩展，左指针收缩跳跃

### 代码

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        char_index = {}  # 记录字符最后出现的位置
        left = 0
        max_len = 0

        for right, char in enumerate(s):
            # 如果字符已存在且在窗口内，移动左边界
            if char in char_index and char_index[char] >= left:
                left = char_index[char] + 1

            char_index[char] = right  # 更新字符位置
            max_len = max(max_len, right - left + 1)

        return max_len
```

## 438. 找到字符串中所有字母异位词 【中等】

### 思路

**固定窗口 + 字符计数**

- 异位词 = 字符种类和个数相同
- 维护长度为 `len(p)` 的固定窗口
- 用数组/哈希表统计窗口内字符频率，与 p 的频率比较
- 窗口滑动：右边加一个字符，左边减一个字符

**优化**: 维护"有效字符数"，避免每次比较整个频率表

### 代码

```python
class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        if len(s) < len(p):
            return []

        result = []
        p_count = [0] * 26
        s_count = [0] * 26

        # 统计 p 的字符频率
        for char in p:
            p_count[ord(char) - ord('a')] += 1

        # 滑动窗口
        for i in range(len(s)):
            # 右边界字符进入窗口
            s_count[ord(s[i]) - ord('a')] += 1

            # 左边界字符离开窗口（窗口大小超过 len(p)）
            if i >= len(p):
                s_count[ord(s[i - len(p)]) - ord('a')] -= 1

            # 比较频率是否相同
            if s_count == p_count:
                result.append(i - len(p) + 1)

        return result
```

## 560. 和为 K 的子数组 【中等】

### 思路

**前缀和 + 哈希表**

- **注意**: 此题不能用滑动窗口，因为数组包含负数
- 前缀和定义: `prefix[i] = nums[0] + nums[1] + ... + nums[i-1]`
- 子数组和 `nums[i..j] = prefix[j+1] - prefix[i]`
- 若 `prefix[j] - prefix[i] = k`，则 `prefix[i] = prefix[j] - k`
- 用哈希表记录每个前缀和出现的次数

**核心**: 找满足 `prefix[j] - k` 的前缀和个数

### 代码

```python
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        count = 0
        prefix_sum = 0
        # 前缀和出现次数，初始化 {0: 1} 表示前缀和为0出现1次
        prefix_count = {0: 1}

        for num in nums:
            prefix_sum += num

            # 查找是否存在 prefix_sum - k
            if prefix_sum - k in prefix_count:
                count += prefix_count[prefix_sum - k]

            # 更新当前前缀和的出现次数
            prefix_count[prefix_sum] = prefix_count.get(prefix_sum, 0) + 1

        return count
```

## 239. 滑动窗口最大值 【困难】

### 思路

**单调递减队列**

- 维护一个**单调递减的双端队列**，存储元素下标
- 队首始终是当前窗口的最大值
- 新元素入队前，将队尾所有比它小的元素弹出（它们不可能成为最大值）
- 检查队首元素是否超出窗口范围，超出则弹出

**核心**: 队列中元素从队首到队尾单调递减，保证队首是最大值

### 代码

```python
from collections import deque

class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        result = []
        dq = deque()  # 存储下标，保持对应值单调递减

        for i in range(len(nums)):
            # 1. 移除队首超出窗口范围的元素
            if dq and dq[0] <= i - k:
                dq.popleft()

            # 2. 维护单调性：移除队尾所有比当前元素小的
            while dq and nums[dq[-1]] < nums[i]:
                dq.pop()

            # 3. 当前元素下标入队
            dq.append(i)

            # 4. 窗口形成后，记录最大值（队首）
            if i >= k - 1:
                result.append(nums[dq[0]])

        return result
```

## 76. 最小覆盖子串 【困难】

### 思路

**滑动窗口 + 双指针**

- 用两个哈希表: `need` 记录目标字符需求，`window` 记录窗口内字符
- 维护 `valid` 表示满足需求的字符种类数
- **扩展**: 右指针移动，将字符加入窗口
- **收缩**: 当 `valid == len(need)` 时，左指针收缩，尝试缩小窗口
- 收缩过程中更新最小覆盖子串

**核心**: 先扩展找到可行解，再收缩优化解

### 代码

```python
from collections import Counter

class Solution:
    def minWindow(self, s: str, t: str) -> str:
        if not t or not s:
            return ""

        need = Counter(t)  # 目标字符需求
        window = {}        # 窗口内字符计数

        left = 0
        valid = 0          # 满足需求的字符种类数
        start = 0          # 最小覆盖子串起始位置
        min_len = float('inf')

        for right, char in enumerate(s):
            # 扩展窗口
            if char in need:
                window[char] = window.get(char, 0) + 1
                if window[char] == need[char]:
                    valid += 1

            # 收缩窗口
            while valid == len(need):
                # 更新最小覆盖子串
                if right - left + 1 < min_len:
                    min_len = right - left + 1
                    start = left

                # 移出左边界字符
                left_char = s[left]
                left += 1
                if left_char in need:
                    if window[left_char] == need[left_char]:
                        valid -= 1
                    window[left_char] -= 1

        return "" if min_len == float('inf') else s[start:start + min_len]
```

```python

    left = 0
    window = {}  # 或其他数据结构
    result = ...

    for right in range(len(s)):

        ...

        # 2. 判断是否需要收缩
        while 需要收缩:
            # 3. 更新结果（如果在收缩时更新）
            ...
            # 4. 收缩窗口，移出 s[left]
            ...

        ...

```

**关键点**:
1. **何时扩展**: 每次迭代右指针都会扩展
2. **何时收缩**: 根据题目条件判断（窗口不满足条件/满足条件需要优化）
3. **何时更新答案**: 根据题目求最大还是最小决定
