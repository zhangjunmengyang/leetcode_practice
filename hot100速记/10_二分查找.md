# LeetCode Hot100 二分查找专题

## 35. 搜索插入位置 【简单】

### 思路
标准二分查找模板题。找到第一个 >= target 的位置。
- 如果找到 target，返回其索引
- 如果未找到，返回应该插入的位置（即 left 指针最终位置）

### 代码
```python
class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums) - 1

        while left <= right:
            mid = left + (right - left) // 2  # 防止溢出
            if nums[mid] == target:
                return mid
            elif nums[mid] < target:
                left = mid + 1  # 目标在右半部分
            else:
                right = mid - 1  # 目标在左半部分

        # left 就是第一个 >= target 的位置
        return left
```

## 74. 搜索二维矩阵 【中等】

### 思路
将二维矩阵看作一维有序数组进行二分查找。
- 矩阵大小 m x n，总共 m*n 个元素
- 一维索引 idx 对应二维坐标：row = idx // n, col = idx % n

### 代码
```python
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        if not matrix or not matrix[0]:
            return False

        m, n = len(matrix), len(matrix[0])
        left, right = 0, m * n - 1

        while left <= right:
            mid = left + (right - left) // 2
            # 将一维索引转换为二维坐标
            row, col = mid // n, mid % n
            val = matrix[row][col]

            if val == target:
                return True
            elif val < target:
                left = mid + 1
            else:
                right = mid - 1

        return False
```

## 34. 在排序数组中查找元素的第一个和最后一个位置 【中等】

### 思路
两次二分查找：
1. 第一次找左边界（第一个等于 target 的位置）
2. 第二次找右边界（最后一个等于 target 的位置）

关键：找左边界时，即使找到 target 也继续向左收缩；找右边界同理。

### 代码
```python
class Solution:
    def searchRange(self, nums: List[int], target: int) -> List[int]:
        def findLeft(nums, target):
            """找第一个等于 target 的位置"""
            left, right = 0, len(nums) - 1
            while left <= right:
                mid = left + (right - left) // 2
                if nums[mid] < target:
                    left = mid + 1
                else:
                    right = mid - 1  # nums[mid] >= target，继续向左找
            # 检查 left 是否越界以及是否等于 target
            if left < len(nums) and nums[left] == target:
                return left
            return -1

        def findRight(nums, target):
            """找最后一个等于 target 的位置"""
            left, right = 0, len(nums) - 1
            while left <= right:
                mid = left + (right - left) // 2
                if nums[mid] <= target:
                    left = mid + 1  # nums[mid] <= target，继续向右找
                else:
                    right = mid - 1
            # 检查 right 是否越界以及是否等于 target
            if right >= 0 and nums[right] == target:
                return right
            return -1

        return [findLeft(nums, target), findRight(nums, target)]
```

## 33. 搜索旋转排序数组 【中等】

### 思路
旋转后数组分成两段有序部分。每次二分时，判断 mid 在哪一段：
1. 若 nums[left] <= nums[mid]，说明左半部分有序
   - 若 target 在 [nums[left], nums[mid]) 内，在左半部分找
   - 否则在右半部分找
2. 否则右半部分有序，同理判断

### 代码
```python
class Solution:
    def search(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums) - 1

        while left <= right:
            mid = left + (right - left) // 2

            if nums[mid] == target:
                return mid

            # 判断哪一半是有序的
            if nums[left] <= nums[mid]:  # 左半部分有序
                # 判断 target 是否在左半部分
                if nums[left] <= target < nums[mid]:
                    right = mid - 1
                else:
                    left = mid + 1
            else:  # 右半部分有序
                # 判断 target 是否在右半部分
                if nums[mid] < target <= nums[right]:
                    left = mid + 1
                else:
                    right = mid - 1

        return -1
```

## 153. 寻找旋转排序数组中的最小值 【中等】

### 思路
最小值是旋转点。比较 nums[mid] 与 nums[right]：
- 若 nums[mid] > nums[right]，最小值在右半部分（不含 mid）
- 若 nums[mid] <= nums[right]，最小值在左半部分（含 mid）

注意：这里用 nums[right] 而非 nums[left] 比较更简洁。

### 代码
```python
class Solution:
    def findMin(self, nums: List[int]) -> int:
        left, right = 0, len(nums) - 1

        while left < right:  # 注意：这里是 < 不是 <=
            mid = left + (right - left) // 2

            if nums[mid] > nums[right]:
                # 最小值在 mid 右边（不含 mid）
                left = mid + 1
            else:
                # 最小值在 mid 左边（含 mid）
                right = mid

        # left == right 时找到最小值
        return nums[left]
```

## 4. 寻找两个正序数组的中位数 【困难】

### 思路
核心思想：在较短数组上二分，将两个数组各分成左右两部分，使得：
1. 左部分元素个数 = 右部分元素个数（或多1个）
2. 左部分最大值 <= 右部分最小值

设在 nums1 中分割点为 i，在 nums2 中分割点为 j = (m+n+1)//2 - i

### 代码
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        # 确保 nums1 是较短的数组
        if len(nums1) > len(nums2):
            nums1, nums2 = nums2, nums1

        m, n = len(nums1), len(nums2)
        # half 是左半部分应该有的元素个数
        half = (m + n + 1) // 2

        left, right = 0, m

        while left <= right:
            # i: nums1 左半部分的元素个数
            # j: nums2 左半部分的元素个数
            i = (left + right) // 2
            j = half - i

            # 获取分割线两侧的四个值
            nums1_left = float('-inf') if i == 0 else nums1[i - 1]
            nums1_right = float('inf') if i == m else nums1[i]
            nums2_left = float('-inf') if j == 0 else nums2[j - 1]
            nums2_right = float('inf') if j == n else nums2[j]

            if nums1_left <= nums2_right and nums2_left <= nums1_right:
                # 找到正确的分割点
                if (m + n) % 2 == 1:
                    # 奇数：中位数是左半部分的最大值
                    return max(nums1_left, nums2_left)
                else:
                    # 偶数：中位数是左半部分最大值和右半部分最小值的平均
                    return (max(nums1_left, nums2_left) + min(nums1_right, nums2_right)) / 2
            elif nums1_left > nums2_right:
                # nums1 左边太大，i 需要左移
                right = i - 1
            else:
                # nums2 左边太大，i 需要右移
                left = i + 1

        return 0.0
```

|------|----------|
| 35. 搜索插入位置 | 标准二分模板，返回 left |
