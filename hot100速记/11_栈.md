## 20. 有效的括号 【简单】

### 思路
使用栈匹配括号：遇到左括号入栈，遇到右括号检查栈顶是否匹配。
- 右括号必须与最近的左括号配对 -> 后进先出 -> 栈
- 用哈希表存储括号对应关系，右括号作为 key

### 代码
```python
class Solution:
    def isValid(self, s: str) -> bool:
        stack = []
        # 右括号 -> 左括号 的映射
        mapping = {')': '(', ']': '[', '}': '{'}

        for char in s:
            if char in mapping:
                # 遇到右括号，检查栈顶是否匹配
                if not stack or stack[-1] != mapping[char]:
                    return False
                stack.pop()
            else:
                # 左括号入栈
                stack.append(char)

        # 栈空则全部匹配成功
        return len(stack) == 0
```

## 155. 最小栈 【中等】

### 思路
使用辅助栈同步存储当前最小值：
- 主栈正常存数据
- 辅助栈栈顶始终保存当前主栈中的最小值
- 每次 push 时，辅助栈压入 min(当前值, 辅助栈顶)

### 代码
```python
class MinStack:
    def __init__(self):
        self.stack = []       # 主栈
        self.min_stack = []   # 辅助栈，存储对应位置的最小值

    def push(self, val: int) -> None:
        self.stack.append(val)
        # 辅助栈压入当前最小值
        if not self.min_stack:
            self.min_stack.append(val)
        else:
            self.min_stack.append(min(val, self.min_stack[-1]))

    def pop(self) -> None:
        self.stack.pop()
        self.min_stack.pop()  # 同步弹出

    def top(self) -> int:
        return self.stack[-1]

    def getMin(self) -> int:
        return self.min_stack[-1]  # O(1) 获取最小值
```

## 394. 字符串解码 【中等】

### 思路
双栈法：数字栈 + 字符串栈
- 遇到数字：累积数字
- 遇到 `[`：当前数字和字符串入栈，重置
- 遇到 `]`：弹出数字和之前的字符串，拼接 = 之前字符串 + 当前字符串 * 数字
- 遇到字母：追加到当前字符串

### 代码
```python
class Solution:
    def decodeString(self, s: str) -> str:
        num_stack = []    # 存储重复次数
        str_stack = []    # 存储之前的字符串
        cur_str = ""      # 当前正在构建的字符串
        cur_num = 0       # 当前数字

        for char in s:
            if char.isdigit():
                # 数字可能多位，累积计算
                cur_num = cur_num * 10 + int(char)
            elif char == '[':
                # 保存当前状态，开始新的一层
                num_stack.append(cur_num)
                str_stack.append(cur_str)
                cur_str = ""
                cur_num = 0
            elif char == ']':
                # 弹出并拼接：之前的字符串 + 当前字符串重复n次
                repeat = num_stack.pop()
                prev_str = str_stack.pop()
                cur_str = prev_str + cur_str * repeat
            else:
                # 字母直接追加
                cur_str += char

        return cur_str
```

## 739. 每日温度 【中等】

### 思路
单调递减栈（存下标）：
- 栈中保持温度递减，遇到更高温度时，弹出所有比它低的
- 弹出时计算等待天数 = 当前下标 - 弹出元素下标
- 存下标而非值，方便计算天数差

### 代码
```python
class Solution:
    def dailyTemperatures(self, temperatures: List[int]) -> List[int]:
        n = len(temperatures)
        answer = [0] * n
        stack = []  # 单调递减栈，存储下标

        for i, temp in enumerate(temperatures):
            # 当前温度比栈顶高，说明找到了栈顶元素的"下一个更高温度"
            while stack and temp > temperatures[stack[-1]]:
                prev_idx = stack.pop()
                answer[prev_idx] = i - prev_idx  # 等待天数
            stack.append(i)

        # 栈中剩余元素没有更高温度，保持默认值0
        return answer
```

## 84. 柱状图中最大的矩形 【困难】

### 思路
单调递增栈 + 哨兵：
- 对每个柱子，找左右两边第一个比它矮的位置，计算以它为高的最大矩形
- 单调递增栈：遇到更矮的柱子时，弹出栈顶计算面积
- 首尾添加高度为0的哨兵，简化边界处理
- 面积 = 高度 * (右边界 - 左边界 - 1)

### 代码
```python
class Solution:
    def largestRectangleArea(self, heights: List[int]) -> int:
        # 添加哨兵：首尾各加一个0，避免边界判断
        heights = [0] + heights + [0]
        stack = []  # 单调递增栈，存储下标
        max_area = 0

        for i, h in enumerate(heights):
            # 当前高度小于栈顶，弹出计算面积
            while stack and h < heights[stack[-1]]:
                cur_height = heights[stack.pop()]
                # 宽度 = 右边界i - 左边界stack[-1] - 1
                width = i - stack[-1] - 1
                max_area = max(max_area, cur_height * width)
            stack.append(i)

        return max_area
```

## 总结

| 题目 | 核心技巧 | 关键点 |
|------|----------|--------|
| 20. 有效的括号 | 栈匹配 | 右括号作key的哈希表 |
| 155. 最小栈 | 辅助栈 | 同步维护最小值 |

| 84. 柱状图最大矩形 | 单调递增栈+哨兵 | 弹出时计算面积 |

**栈的核心应用场景**：
1. **括号匹配**：后进先出的特性
2. **单调栈**：寻找下一个更大/更小元素
3. **表达式解析**：保存中间状态，遇到特定符号时处理
