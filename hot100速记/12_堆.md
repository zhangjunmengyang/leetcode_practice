## 215. 数组中的第K个最大元素 【中等】

### 思路

**核心思想：小顶堆维护前K大元素**

1. 维护一个大小为 K 的小顶堆
2. 遍历数组，堆满后只有比堆顶大的元素才能入堆（先弹出堆顶再入堆）
3. 最终堆顶就是第 K 大元素

**为什么用小顶堆？** 小顶堆堆顶是最小值，堆中始终保持最大的 K 个元素，堆顶即为第 K 大。

**时间复杂度：** O(n log k)，每个元素最多一次入堆操作

### 代码

```python
import heapq
from typing import List

class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        # 小顶堆，维护前k大的元素
        heap = []

        for num in nums:
            if len(heap) < k:
                # 堆未满，直接入堆
                heapq.heappush(heap, num)
            elif num > heap[0]:
                # 比堆顶大，替换堆顶
                heapq.heapreplace(heap, num)

        # 堆顶即为第k大元素
        return heap[0]
```

## 347. 前 K 个高频元素 【中等】

### 思路

**核心思想：小顶堆维护前K高频元素**

1. 先用哈希表统计每个元素的出现频率
2. 维护一个大小为 K 的小顶堆，按频率排序
3. 堆中始终保持频率最高的 K 个元素

**关键点：** Python 的 heapq 是小顶堆，堆中存储 (频率, 元素) 元组，自动按频率排序

**时间复杂度：** O(n log k)

### 代码

```python
import heapq
from typing import List
from collections import Counter

class Solution:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:
        # 统计频率
        count = Counter(nums)

        # 小顶堆维护前k高频元素，存储 (频率, 元素)
        heap = []

        for num, freq in count.items():
            if len(heap) < k:
                heapq.heappush(heap, (freq, num))
            elif freq > heap[0][0]:
                # 频率更高，替换堆顶
                heapq.heapreplace(heap, (freq, num))

        # 提取结果（只需要元素，不需要频率）
        return [item[1] for item in heap]
```

## 295. 数据流的中位数 【困难】

### 思路

**核心思想：大顶堆 + 小顶堆，对半分**

1. **大顶堆 max_heap**：存储较小的一半数字（取负实现大顶堆）
2. **小顶堆 min_heap**：存储较大的一半数字
3. **平衡规则**：max_heap 的元素数量 = min_heap 或 min_heap + 1

**添加元素流程：**
- 先加入 max_heap，再把 max_heap 的最大值移到 min_heap
- 若 min_heap 元素更多，把 min_heap 最小值移回 max_heap
- 保证 max_heap 元素数 >= min_heap 元素数

**获取中位数：**
- 奇数个：max_heap 堆顶
- 偶数个：两个堆顶的平均值

**时间复杂度：** 添加 O(log n)，获取中位数 O(1)

### 代码

```python
import heapq

class MedianFinder:

    def __init__(self):
        # 大顶堆存较小一半（取负模拟）
        self.max_heap = []
        # 小顶堆存较大一半
        self.min_heap = []

    def addNum(self, num: int) -> None:
        # 先加入大顶堆（取负）
        heapq.heappush(self.max_heap, -num)

        # 把大顶堆最大值移到小顶堆
        heapq.heappush(self.min_heap, -heapq.heappop(self.max_heap))

        # 平衡：保证大顶堆元素数 >= 小顶堆
        if len(self.min_heap) > len(self.max_heap):
            heapq.heappush(self.max_heap, -heapq.heappop(self.min_heap))

    def findMedian(self) -> float:
        if len(self.max_heap) > len(self.min_heap):
            # 奇数个，大顶堆堆顶
            return -self.max_heap[0]
        else:
            # 偶数个，两堆顶平均值
            return (-self.max_heap[0] + self.min_heap[0]) / 2
```

## 堆专题总结

| 题目 | 核心技巧 | 堆类型 |
|-----|---------|-------|
| 215. 第K大元素 | 小顶堆维护前K大 | 小顶堆 |
| 347. 前K高频 | 统计频率 + 小顶堆维护前K高频 | 小顶堆 |
| 295. 数据流中位数 | 双堆对半分，大顶堆+小顶堆 | 大顶堆+小顶堆 |

**记忆口诀：**
- 求第 K 大/前 K 大 → 小顶堆（堆顶最小，保留最大的 K 个）
- 求第 K 小/前 K 小 → 大顶堆（堆顶最大，保留最小的 K 个）
- Python 大顶堆技巧：存负数
